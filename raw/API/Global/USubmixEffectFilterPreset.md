### USubmixEffectFilterPreset


USubmixEffectFilterPreset
Class which processes audio streams and uses parameters defined in the preset class.

**属性**:

- `FSubmixEffectFilterSettings Settings`


**方法**:

- `SetFilterAlgorithm(ESubmixFilterAlgorithm InAlgorithm)`  
  Sets the filter algorithm
- `SetFilterCutoffFrequency(float32 InFrequency)`  
  Sets the base filter cutoff frequency
- `SetFilterCutoffFrequencyMod(float32 InFrequency)`  
  Sets the mod filter cutoff frequency
- `SetFilterQ(float32 InQ)`  
  Sets the filter Q
- `SetFilterQMod(float32 InQ)`  
  Sets the filter Q
- `SetFilterType(ESubmixFilterType InType)`  
  Sets the filter type
- `SetSettings(FSubmixEffectFilterSettings InSettings)`  
  Set all filter effect settings

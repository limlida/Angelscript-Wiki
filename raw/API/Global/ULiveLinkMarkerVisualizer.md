### ULiveLinkMarkerVisualizer


An Instanced Static Mesh Component to represent Motion Capture marker data locations

**属性**:

- `UMaterialInstanceDynamic DynamicMaterial [Dynamic material to use on the instance static meshes.]`
- `FLiveLinkSubjectName LiveLinkSubject [Live Link subject to drive the instanced static mesh transforms.]`
- `TArray<FTransform> MarkerLocations [Array of Transforms for the markers.]`
- `EAxisSign XAxisSign [Sign for the x axis of the marker.]`
- `EAxisSign YAxisSign [Sign for the y axis of the marker.]`
- `bool bDrawLabels [To Do - add text labels to drawn markers]`
- `bool bEvaluateLiveLink [Bool to control evaluation of Live Link data.]`

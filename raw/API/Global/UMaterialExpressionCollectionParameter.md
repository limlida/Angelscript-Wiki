### UMaterialExpressionCollectionParameter


**属性**:

- `UMaterialParameterCollection Collection [The Parameter Collection to use.]`
- `FName Group [The name of the parameter Group to display in MaterialInstance Editor. Default is None group]`
- `FName ParameterName [Name of the parameter being referenced.]`
- `int SortPriority [Controls where the this parameter is displayed in a material instance parameter list. The lower the number the higher up in the parameter list.]`

### UPatternTool_RadialSettings


Settings for Radial Patterns in the Pattern Tool

**属性**:

- `float AngleShift [Fixed offset added to Start/End Angles]`
- `int Count [Number of  Pattern Elements to place]`
- `float EndAngle [End angle of the Circle/Arc]`
- `float Radius [Radius of the Circle/Arc]`
- `EPatternToolAxisSpacingMode SpacingMode [Spacing Technique used to distribute Pattern Elements around the Circle/Arc]`
- `float StartAngle [Start angle of the Circle/Arc]`
- `float StepSize [Fixed Increment (in Degrees) used to position Pattern Elements around the Circle/Arc]`
- `bool bOriented [If true, Pattern elements are rotated to align with the Circle tangent]`

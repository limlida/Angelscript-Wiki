### UParticleModuleKillHeight


**属性**:

- `FRawDistributionFloat Height [The height at which to kill the particle.]`
- `bool bAbsolute [If true, the height should be treated as a world-space position.]`
- `bool bApplyPSysScale [If true, take the particle systems scale into account]`
- `bool bFloor [If true, the plane should be considered a floor - ie kill anything BELOW it.
If false, if is a ceiling - ie kill anything ABOVE it.]`

### UGameFeatureAction_DataRegistrySource


Specifies a list of source assets to add to Data Registries when this feature is activated

**属性**:

- `TArray<FDataRegistrySourceToAdd> SourcesToAdd [List of sources to add when this feature is activated]`
- `bool bPreloadInEditor [If true, this will preload the sources when the feature is registered in the editor to support the editor pickers]`

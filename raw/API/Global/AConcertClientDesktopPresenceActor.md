### AConcertClientDesktopPresenceActor


A AConcertClientDesktopPresenceActor is a child of AConcertClientPresenceActor that is used to represent users in desktop

**属性**:

- `UStaticMeshComponent DesktopMeshComponent [The camera mesh component to show visually where the camera is placed]`
- `USplineMeshComponent LaserPointer [Spline mesh representing laser]`

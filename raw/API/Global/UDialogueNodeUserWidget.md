### UDialogueNodeUserWidget


Parent class for a custom UMG widget that narrative will add to dialogue nodes if you want to override narratives default UI

- `UDialogue Dialogue []`
- `UVerticalBox LeftPinBox []`
- `UDialogueNode Node []`

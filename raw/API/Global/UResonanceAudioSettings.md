### UResonanceAudioSettings


**属性**:

- `FSoftObjectPath GlobalReverbPreset [Default settings for global reverb: This is overridden when a player enters Audio Volumes.]`
- `FSoftObjectPath GlobalSourcePreset [Default settings for global source settings]`
- `FSoftObjectPath OutputSubmix [Reference to submix where reverb plugin audio is routed.]`
- `ERaQualityMode QualityMode [Global Quality mode to use for directional sound sources.]`

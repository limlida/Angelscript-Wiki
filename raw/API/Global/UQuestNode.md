### UQuestNode


Base class for states and branches in the quests state machine

**属性**:

- `FText Description [Description for this quest node. For example "Kill 10 Goblins", "Find the Gemstone", or "I've found the Gemstone, I need to return to King Edward"]`
- `FName OnEnteredFuncName [Name of custom event to call when this state/branch is reached/taken]`

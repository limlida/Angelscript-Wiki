### UMaterialEditorOptions


**属性**:

- `TArray<FString> FavoriteExpressions [The users favorite material expressions.]`
- `bool bAlwaysRefreshAllPreviews [If true, always refresh all expression previews.]`
- `bool bHideUnrelatedNodes [If true, fade nodes which are not connected to the selected nodes]`
- `bool bHideUnusedConnectorsSetting [If true, don't render connectors that are not connected to anything.]`
- `bool bLivePreviewUpdate [If true, always refresh the material preview.]`
- `bool bRealtimeExpressionViewport [If true, the linked object viewport updates in realtime.]`
- `bool bRealtimeMaterialViewport [If true, the 3D material preview viewport updates in realtime.]`
- `bool bUseUnsortedMenus [If false, use expression categorized menus.]`

### UMovieSceneByteTrack


Handles manipulation of byte properties in a movie scene

**方法**:

- `UEnum GetByteTrackEnum()`  
  Get this byte track's enum

@param Track        The track to use
@return The enum for this byte track
- `SetByteTrackEnum(UEnum InEnum)`  
  Set this byte track's enum

@param Track        The track to use
@param InEnum The enum to set

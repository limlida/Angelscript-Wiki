### UBlockAbilityTagsGameplayEffectComponent


Handles blocking the activation of Gameplay Abilities based on Gameplay Tags for the Target actor of the owner Gameplay Effect

**属性**:

- `FInheritedTagContainer InheritableBlockedAbilityTagsContainer [These tags are applied to the target actor of the Gameplay Effect.  Blocked Ability Tags prevent Gameplay Abilities with these tags from executing.]`

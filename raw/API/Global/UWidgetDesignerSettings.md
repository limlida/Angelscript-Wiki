### UWidgetDesignerSettings


Implements the settings for the Widget Blueprint Designer.

**属性**:

- `EDisplayOnCompile CreateOnCompile [Setting to automatically create compile tab based on compile results]`
- `FUintVector2 DefaultPreviewResolution [The default preview resolution in the designer.]`
- `EDisplayOnCompile DismissOnCompile [Setting to automatically dismiss compile tab based on compile results]`
- `bool GridSnapEnabled [If enabled, actor positions will snap to the grid.]`
- `bool bExecutePreConstructEvent [Should the designer run the design event?  Disable this if you're seeing crashes in the designer,
you may have some unsafe code running in the designer.]`
- `bool bLockToPanelOnDragByDefault []`
- `bool bRespectLocks [Should the designer respect locked widgets?  If true, the designer by default
will not allow you to select locked widgets in the designer view.]`
- `bool bShowOutlines [Should the designer show outlines by default?]`

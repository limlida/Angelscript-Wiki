### UAudioInsightsEditorSettings


**属性**:

- `FAudioEventLogSettings EventLogSettings [Settings for the Event Log]`
- `FLoudnessMeterRackUnitSettings LoudnessMeterSettings [Settings for analyzer rack loudness meter widget]`
- `FSoundDashboardSettings SoundDashboardSettings [Settings for the Sound Dashboard]`
- `FSpectrogramRackUnitSettings SpectrogramSettings [Settings for analyzer rack spectrogram widget]`
- `FSpectrumAnalyzerRackUnitSettings SpectrumAnalyzerSettings [Settings for analyzer rack spectrum analyzer widget]`
- `bool bWorldFilterDefaultsToFirstClient [Whether to automatically set the first PIE client in Audio Insights World Filter.]`

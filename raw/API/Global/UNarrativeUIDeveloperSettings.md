### UNarrativeUIDeveloperSettings


Base class for Narrative UI settings, like global styling, etc.

**属性**:

- `FLinearColor UIInvertAccentColor [Invert Accent color UI elements will set themselves to.]`
- `FLinearColor UIInvertColor [Invert color UI elements will set themselves to.]`
- `FLinearColor UIPrimaryColor [Primary color UI elements will set themselves to.]`

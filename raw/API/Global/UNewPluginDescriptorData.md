### UNewPluginDescriptorData


An object used internally by the New Plugin Wizard to set user-defined plugin descriptor information.
This should not be used outside of the New Plugin Wizard dialog

**属性**:

- `FString CreatedBy [The author of this content]`
- `FString CreatedByURL [Optional hyperlink for the author's website]`
- `FString Description [A description for this content]`
- `bool bIsBetaVersion [Marks this content as being in beta]`

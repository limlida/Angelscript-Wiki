### URigVMEditorSettings


Customize RigVM Rig Editor.

**属性**:

- `bool bAutoLinkMutableNodes [When this is checked mutable nodes (nodes with an execute pin)
will be hooked up automatically.]`
- `bool bEnableContextMenuTimeSlicing [When the Blueprint graph context menu is invoked (e.g. by right-clicking in the graph or dragging off a pin), do not block the UI while populating the available actions list.]`
- `bool bFadeOutUnrelatedNodes [fade out unrelated nodes]`
- `bool bHighlightSimilarNodes [with this checked similar nodes in the graph will be highlighted.
this highlighting can then also be used to functionize items]`
- `bool bUseFlashLight [use a flash light to brighten up nodes]`

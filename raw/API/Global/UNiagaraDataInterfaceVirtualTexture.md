### UNiagaraDataInterfaceVirtualTexture


Data Interface allowing sampling of a texture

**属性**:

- `URuntimeVirtualTexture Texture [, meta = (AllowedClasses = "/Script/Engine.VirtualTexture2D"))]`
- `FNiagaraUserParameterBinding TextureUserParameter [When valid the user parameter is used as the texture rather than the one on the data interface]`

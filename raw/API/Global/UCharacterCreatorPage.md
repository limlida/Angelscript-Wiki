### UCharacterCreatorPage


Defines a page in the character creator

**属性**:

- `FTransform PageCameraTransform [The transform in the level the camera should lerp to when this page is selected]`
- `FGameplayTag PageID [The gameplay tag identifying this creator section]`
- `TArray<TObjectPtr<UCharacterCreatorSection>> PageSections [All of the sections on the page]`
- `FText PageTitleText [The display name for this page]`

### UNarrativeDeveloperSettings


Local developer settings per user. Contains useful tweaks devs may want locally, but don't want to propogate to other users

**属性**:

- `TArray<TObjectPtr<UNPCDefinition>> NPCAllowList [We'll only allow NPCs in this list to spawn. This can be useful for checking]`
- `TArray<TObjectPtr<UPlayerDefinition>> PlayerDefinitionOverrides [We'll use this as the player definition instead of the game mode specified definition]`

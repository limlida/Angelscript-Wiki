### UReflectionCaptureComponent


-> will be exported to EngineDecalClasses.h

**属性**:

- `float32 Brightness [A brightness control to scale the captured scene's reflection intensity.]`
- `FVector CaptureOffset [World space offset to apply before capturing.]`
- `UTextureCube Cubemap [Cubemap to use for reflection if ReflectionSourceType is set to RS_SpecifiedCubemap.]`
- `EReflectionSourceType ReflectionSourceType [Indicates where to get the reflection source from.]`
- `float32 SourceCubemapAngle [Angle to rotate the source cubemap when SourceType is set to SLS_SpecifiedCubemap.]`

### UMetaHumanSDKSettings


Project Settings for the MetaHuman SDK

**属性**:

- `FDirectoryPath CharacterAssemblyPackagingPath [The asset path for finding MetaHuman Character Assemblies for Packaging.]`
- `FDirectoryPath CharacterAssetPackagingPath [The asset path for finding Editable MetaHuman Character Assets for Packaging.]`
- `FDirectoryPath CinematicImportPath [The asset path for importing Cinematic MetaHumans.]`
- `FDirectoryPath GroomPackagingPath [The asset path for finding MetaHuman Groom Assets for Packaging.]`
- `FDirectoryPath OptimizedImportPath [The asset path for importing Optimized MetaHumans.]`
- `FDirectoryPath OutfitPackagingPath [The asset path for finding MetaHuman Outfit-based Clothing Assets for Packaging.]`
- `FDirectoryPath SkeletalClothingPackagingPath [The asset path for finding MetaHuman Skeletal Clothing Assets for Packaging.]`

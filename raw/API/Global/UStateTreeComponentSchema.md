### UStateTreeComponentSchema


StateTree for Actors with StateTree component.

**属性**:

- `TSubclassOf<AActor> ContextActorClass [Actor class the StateTree is expected to run on. Allows to bind to specific Actor class' properties.]`
- `EStateTreeComponentSchemaScheduledTickPolicy ScheduledTickPolicy [Indicates if the execution can sleep and the tick delayed.
The default value set by the cvar StateTree.Component.DefaultScheduledTickAllowed]`

### UOnlinePIESettings


Setup login credentials for the Play In Editor (PIE) feature

**属性**:

- `TArray<FOnlineAccountStoredCredentials> Logins [Login credentials, at least one for each instance of PIE that is intended to be run]`
- `bool bOnlinePIEEnabled [Attempt to login with user credentials on a backend service before launching the PIE instance.]`

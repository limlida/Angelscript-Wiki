### UPlasticSourceControlProjectSettings


Project Settings for Unity Version Control (formerly Plastic SCM). Saved in Config/DefaultEditor.ini

**属性**:

- `int LimitNumberOfRevisionsInHistory [If a non-null value is set, limit the maximum number of revisions requested to Unity Version Control to display in the "History" window.]`
- `TMap<FString,FString> UserNameToDisplayName [Map Unity Version Control user names (typically e-mail addresses or company domain names) to display names for brevity.]`
- `bool bHideEmailDomainInUsername [Hide the domain part of an username e-mail address (eg @gmail.com) if the UserNameToDisplayName map didn't match (enabled by default).]`
- `bool bPromptForCheckoutOnChange [If enabled, you'll be prompted to check out changed files (enabled by default). Checkout is needed to work with Changelists.]`
- `bool bShowBranchCommentColumn [Show the comment of the branch]`
- `bool bShowBranchCreatedByColumn [Show the name of the creator of the branch]`
- `bool bShowBranchDateColumn [Show the date of creation of the branch]`
- `bool bShowBranchRepositoryColumn [Show the repository where the branch is created (hidden by default)]`
- `bool bShowChangesetBranchColumn [Show the branch the changeset was created on]`
- `bool bShowChangesetCommentColumn [Show the comment of the changeset]`
- `bool bShowChangesetCreatedByColumn [Show the name of the creator of the changeset]`
- `bool bShowChangesetDateColumn [Show the date of creation of the changeset]`
- `bool bShowLockDateColumn [Show the Date of the lock]`
- `bool bShowLockDestinationBranchColumn [Show the Destination Branch of the lock (hidden by default)]`
- `bool bShowLockIdColumn [Show the Id of the lock (hidden by default)]`
- `bool bShowLockWorkspaceColumn [Show the Workspace of the lock]`

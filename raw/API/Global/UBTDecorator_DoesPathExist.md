### UBTDecorator_DoesPathExist


Does Path Exist decorator node.
A decorator node that bases its condition on whether a path exists between two points from the Blackboard.

**属性**:

- `FBlackboardKeySelector BlackboardKeyA [blackboard key selector]`
- `FBlackboardKeySelector BlackboardKeyB [blackboard key selector]`
- `FValueOrBBKey_Class FilterClass ["None" will result in default filter being used]`
- `FValueOrBBKey_Enum PathQueryType []`

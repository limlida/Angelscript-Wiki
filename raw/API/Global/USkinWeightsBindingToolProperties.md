### USkinWeightsBindingToolProperties


**属性**:

- `ESkinWeightsBindType BindingType [Binding type to use]`
- `int MaxInfluences [Maximum bones that will influence each vertex]`
- `float32 Stiffness [Stiffness of binding. Lower values allow more distant bones to contribute more]`
- `int VoxelResolution [The resolution of the voxel grid if doing geodesic voxel binding]`

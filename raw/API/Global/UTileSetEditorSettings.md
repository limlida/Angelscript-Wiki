### UTileSetEditorSettings


Settings for the Paper2D tile set editor

**属性**:

- `FColor DefaultBackgroundColor [Default background color for new tile set assets]`
- `int ExtrusionAmount [The amount to extrude out from the edge of each tile (in pixels)]`
- `bool bFillWithTransparentBlack [Should we use transparent black or white when filling the texture areas that aren't covered by tiles?]`
- `bool bPadToPowerOf2 [Should we pad the conditioned texture to the next power of 2?]`
- `bool bShowGridByDefault [Should the grid be shown by default when the editor is opened?]`

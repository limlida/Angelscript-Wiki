### UCharacterCreatorOption_Scalar


A global scalar value option that can be changed in the character creator - could be anything, height, pupil size, any sort of scalar you require.

**属性**:

- `float32 DefaultValue [The default value the scalar will be set to]`
- `float32 MaxValue [The max value this scalar can go to]`
- `float32 MinValue [The min value this scalar can go to]`
- `FGameplayTag ScalarTagID [The ID you can use to reference this scalar value]`
- `float32 StepValue [The scalars UI slider will use this step size]`

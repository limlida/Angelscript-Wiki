### UUVEditorBrushSelectToolProperties


**属性**:

- `float32 LivePreviewBrushRadius [Radius of the brush in the 3D live preview view.]`
- `float32 UnwrapBrushRadius [Radius of the brush in the 2D UV unwrap view.]`
- `bool bClearSelectionOnEachDrag [When true, each drag will clear the existing selection if neither Shift (add to selection) nor
 Ctrl (remove from selection) nor both (toggle selection) are pressed. When false, selection
 will not clear, requiring manual removal from selection using Ctrl.]`
- `bool bExpandToIslands [When true, brush selects whole UV islands instead of individual triangles.]`

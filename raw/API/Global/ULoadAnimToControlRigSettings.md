### ULoadAnimToControlRigSettings


settings used when loading animation sequences into a control rig section

**属性**:

- `FFrameNumber EndFrame []`
- `FSmartReduceParams SmartReduce []`
- `FFrameNumber StartFrame []`
- `bool bOntoSelectedControls [Load animation onto only selected controls]`
- `bool bReduceKeys [Reduce Keys]`
- `bool bResetControls [Reset controls to initial value on every frame]`
- `bool bUseCustomTimeRange []`

### UMaterialExpressionMaterialXFractal3D


Zero-centered 3D Fractal noise in 1, 2, 3 or 4 channels, created by summing several
octaves of 3D Perlin noise, increasing the frequency and decreasing the amplitude at each octave.

**属性**:

- `float32 ConstAmplitude [only used if Amplitude is not hooked up]`
- `float32 ConstDiminish [only used if Diminish is not hooked up]`
- `float32 ConstLacunarity [only used if Lacunarity is not hooked up]`
- `int ConstOctaves [only used if Octaves is not hooked up]`

### UAudioEditorSettings


**属性**:

- `FString AudioMixerModuleName [The device id of the currently selected audio output device. Requires editor restart to take effect]`
- `FString AudioOutputDeviceId [The device id of the currently selected audio output device.]`
- `FName MenuPosition [Specify placement of sound menus when right-clicking on sound asset types in Content Browser (ex. 'Last' will sort to bottom, 'First' to top).]`
- `bool bPinSoundAttenuationInAssetMenu [Whether to pin the Sound Attenuation asset type when creating new assets. Requires editor restart to take effect.]`
- `bool bPinSoundConcurrencyInAssetMenu [Whether to pin the Sound Concurrency asset type when creating new assets. Requires editor restart to take effect.]`
- `bool bPinSoundCueInAssetMenu [Whether to pin the Sound Cue asset type when creating new assets. Requires editor restart to take effect.]`
- `bool bPinSoundCueTemplateInAssetMenu [Whether to pin the Sound Cue Template asset type when creating new assets. Requires editor restart to take effect.]`
- `bool bUseAudioAttenuation [Whether or not should Audio Attenuation be used by default, for Non-Game Worlds]`
- `bool bUseSubmixesForPreviewSound [Use the submix tree when playing preview sounds]`
- `bool bUseSystemDevice [Use current audio playback device selected in the operating system.]`

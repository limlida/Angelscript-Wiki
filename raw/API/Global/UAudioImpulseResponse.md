### UAudioImpulseResponse


UAudioImpulseResponse
UAsset used to represent Imported Impulse Responses

**属性**:

- `float32 NormalizationVolumeDb [Used to account for energy added by convolution with "loud" Impulse Responses.]`
- `bool bTrueStereo [If true, impulse response channels are interpreted as true stereo which allows channel crosstalk. If false, impulse response channels are interpreted as independent channel impulses.]`

### UEdGraphNode_Comment


**属性**:

- `FLinearColor CommentColor [Color to style comment with]`
- `int FontSize [Size of the text in the comment box]`
- `ECommentBoxMode MoveMode [Whether the comment should move any fully enclosed nodes around when it is moved]`
- `FText NodeDetails [Details field if more info is needed to be communicated (will show up in tooltip)]`
- `bool bColorCommentBubble [Whether to use Comment Color to color the background of the comment bubble shown when zoomed out.]`
- `bool bCommentBubbleVisible_InDetailsPanel [Whether to show a zoom-invariant comment bubble when zoomed out (making the comment readable at any distance).]`

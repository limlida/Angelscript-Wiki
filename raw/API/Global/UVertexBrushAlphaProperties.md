### UVertexBrushAlphaProperties


Tool Properties for a brush alpha mask

**属性**:

- `UTexture2D Alpha [Alpha mask applied to brush stamp. Red channel is used.]`
- `float32 RandomRange [Bounds of random generation (positive and negative) for randomized stamps]`
- `float32 RotationAngle [Alpha is rotated by this angle, inside the brush stamp frame (vertically aligned)]`
- `bool bRandomize [If true, a random angle in +/- RandomRange is added to Rotation angle for each stamp]`

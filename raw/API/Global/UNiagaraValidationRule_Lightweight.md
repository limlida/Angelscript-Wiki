### UNiagaraValidationRule_Lightweight


Validation rule to check for lightweight usage.

**属性**:

- `FNiagaraPlatformSet Platforms [Platforms this validation rule will apply to.]`
- `TOptional<ENiagaraValidationSeverity> UsedWithEmitter [When set if we have an emitter present it will be flagged at this severity]`
- `TOptional<ENiagaraValidationSeverity> UsingExperimentalModule [When set if an experimental module is found it will be flagged at this severity]`

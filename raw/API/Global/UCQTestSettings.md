### UCQTestSettings


Implements per project Engine settings for the CQTest plugin.

**属性**:

- `float32 CommandTimeout [Timeout for WaitUntil latent actions.]`
- `float32 MapTestTimeout [Timeout for `FMapTestSpawner::AddWaitUntilLoadedCommand` latent action used during UWorld loading.]`
- `float32 NetworkTimeout [Timeout for WaitUntil latent actions from the PIENetworkComponent.]`

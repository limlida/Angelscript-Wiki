### UPaperExtractSpritesSettings


**属性**:

- `FLinearColor BackgroundColor [The viewport background color]`
- `int NamingStartIndex [The number to start naming with]`
- `FString NamingTemplate [The name of the sprite that will be created. {0} will get replaced by the sprite number.]`
- `FLinearColor OutlineColor [The color of the sprite boundary outlines]`
- `ESpriteExtractMode SpriteExtractMode [Sprite extract mode]`
- `FLinearColor ViewportTextureTint [Apply a tint to the texture in the viewport to improve outline visibility in this editor]`

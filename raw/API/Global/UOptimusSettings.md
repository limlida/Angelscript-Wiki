### UOptimusSettings


**属性**:

- `TSoftObjectPtr<UMeshDeformer> DefaultDeformer [A default deformer that will be used on a skinned mesh if no other deformer has been set.]`
- `EOptimusDefaultDeformerMode DefaultMode [Set when skinned meshes should have a default deformer applied.]`
- `TSoftObjectPtr<UMeshDeformer> DefaultRecomputeTangentDeformer [A default deformer that will be used on a skinned mesh if no other deformer has been set, and if the mesh has requested to recompute tangets.]`

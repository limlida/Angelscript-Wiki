### UGizmoElementLineBase


Base class for 2d and 3d primitive objects which support line drawing,
intended to be used as part of 3D Gizmos.

**属性**:

- `float32 HoverLineThicknessMultiplier [Multiplier applied to line thickness when hovering]`
- `float32 InteractLineThicknessMultiplier [Multiplier applied to line thickness when interacting]`
- `float32 SelectLineThicknessMultiplier [Multiplier applied to line thickness when selected]`
- `float32 SubdueLineThicknessMultiplier [Multiplier applied to line thickness when subdued]`

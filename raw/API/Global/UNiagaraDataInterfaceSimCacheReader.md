### UNiagaraDataInterfaceSimCacheReader


Data interface to read properties from a Niagara Simulation Cache

**属性**:

- `FName EmitterBinding [Which Emitter we should read from within the SimCache.]`
- `UNiagaraSimCache SimCache [Optional source SimCache to use, if the user parameter binding is valid this will be ignored.]`
- `FNiagaraUserParameterBinding SimCacheBinding [User parameter Object binding if this is not a valid sim cache the default one will be used instead.]`

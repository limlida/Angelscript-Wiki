### UMaterialExpressionSpeedTree

- `ESpeedTreeLODType LODType [The type of LOD to use]`
- `ESpeedTreeWindType WindType [The type of wind effect used on this tree. This can only go as high as it was in the SpeedTree Modeler, but you can set it to a lower option for lower quality wind and faster rendering.]`

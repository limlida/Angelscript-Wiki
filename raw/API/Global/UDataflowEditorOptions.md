### UDataflowEditorOptions


A settings class used to store and retreive user settings related to the Dataflow Editor

**属性**:

- `EDataflowConstructionViewportMousePanButton ConstructionViewportMousePanButton [Which mouse button controls camera panning in the Construction Viewport in 2D view mode]`
- `EDataflowEditorEvaluationMode EditorEvaluationMode []`

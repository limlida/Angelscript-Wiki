### UAISense


**属性**:

- `EAISenseNotifyType NotifyType`
- `bool bAutoRegisterAllPawnsAsSources [If true all newly spawned pawns will get auto registered as source for this sense.]`
- `bool bWantsNewPawnNotification [whether this sense is interested in getting notified about new Pawns being spawned
    this can be used for example for automated sense sources registration]`

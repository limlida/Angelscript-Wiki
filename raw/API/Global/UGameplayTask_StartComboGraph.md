### UGameplayTask_StartComboGraph


Convenience task for running combo graphs assets.

**属性**:

- `FComboGraphTaskEventDelegate__GameplayTask_StartComboGraph EventReceived [Event called when the owner receives a gameplay event combo graph is listening to]`
- `FComboGraphTaskEventDelegate__GameplayTask_StartComboGraph OnActivationFailed [Event called when ability failed to activate]`
- `FComboGraphTaskEventDelegate__GameplayTask_StartComboGraph OnActivationSuccess [Event called when ability was successfully activated]`
- `FComboGraphTaskEventDelegate__GameplayTask_StartComboGraph OnGraphEnd [Event called when ability (and combo graph) ends]`

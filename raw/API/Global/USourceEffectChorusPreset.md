### USourceEffectChorusPreset


**属性**:

- `FSourceEffectChorusSettings Settings`


**方法**:

- `SetDepth(float32 Depth)`
- `SetDepthModulator(const USoundModulatorBase Modulator)`
- `SetDepthModulators(TSet<USoundModulatorBase> Modulators)`
- `SetDry(float32 DryAmount)`
- `SetDryModulator(const USoundModulatorBase Modulator)`
- `SetDryModulators(TSet<USoundModulatorBase> Modulators)`
- `SetFeedback(float32 Feedback)`
- `SetFeedbackModulator(const USoundModulatorBase Modulator)`
- `SetFeedbackModulators(TSet<USoundModulatorBase> Modulators)`
- `SetFrequency(float32 Frequency)`
- `SetFrequencyModulator(const USoundModulatorBase Modulator)`
- `SetFrequencyModulators(TSet<USoundModulatorBase> Modulators)`
- `SetModulationSettings(FSourceEffectChorusSettings ModulationSettings)`
- `SetSettings(FSourceEffectChorusBaseSettings Settings)`  
  Sets just base (i.e. carrier) setting values without modifying modulation source references
- `SetSpread(float32 Spread)`
- `SetSpreadModulator(const USoundModulatorBase Modulator)`
- `SetSpreadModulators(TSet<USoundModulatorBase> Modulators)`
- `SetWet(float32 WetAmount)`
- `SetWetModulator(const USoundModulatorBase Modulator)`
- `SetWetModulators(TSet<USoundModulatorBase> Modulators)`

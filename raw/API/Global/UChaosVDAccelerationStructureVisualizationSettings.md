### UChaosVDAccelerationStructureVisualizationSettings


Settings object that stores the values that control how acceleration structures are debug drawn

**属性**:

- `float32 BaseThickness [The base thickness used to draw node bounds]`
- `ESceneDepthPriorityGroup DepthPriority [The depth priority used for while drawing contact data. Can be World or Foreground (with this one the shapes will be drawn on top of the geometry and be always visible)]`

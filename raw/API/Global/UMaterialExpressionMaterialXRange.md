### UMaterialExpressionMaterialXRange


A material expression that Remap a value from one range to another, optionally
applying a gamma correction in the middle, and optionally clamping output values.

**属性**:

- `float32 ConstGamma []`
- `float32 ConstInputHigh []`
- `float32 ConstInputLow []`
- `float32 ConstTargetHigh []`
- `float32 ConstTargetLow []`
- `bool bConstClamp []`

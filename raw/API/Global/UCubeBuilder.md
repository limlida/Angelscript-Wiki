### UCubeBuilder


**属性**:

- `bool Hollow [Whether this is a hollow or solid cube]`
- `bool Tessellated [Whether extra internal faces should be generated for each cube face]`
- `float32 WallThickness [The thickness of the cube wall when hollow]`
- `float32 X [The size of the cube in the X dimension]`
- `float32 Y [The size of the cube in the Y dimension]`
- `float32 Z [The size of the cube in the Z dimension]`

### UPaperExtractSpriteGridSettings


**属性**:

- `int CellHeight [The height of each sprite in grid mode]`
- `int CellWidth [The width of each sprite in grid mode]`
- `int MarginX [Margin from the left of the texture to the first sprite]`
- `int MarginY [Margin from the top of the texture to the first sprite]`
- `int NumCellsX [Number of cells extracted horizontally. Can be used to limit the number of sprites extracted. Set to 0 to extract all sprites]`
- `int NumCellsY [Number of cells extracted vertically. Can be used to limit the number of sprites extracted. Set to 0 to extract all sprites]`
- `int SpacingX [Horizontal spacing between sprites]`
- `int SpacingY [Vertical spacing between sprites]`

### UBrushBaseProperties


Standard properties for a Brush-type Tool

**属性**:

- `float32 BrushFalloffAmount [Amount of falloff to apply (0.0 - 1.0)]`
- `float32 BrushRadius [Radius of brush]`
- `float32 BrushSize [Relative size of brush]`
- `float32 BrushStrength [Strength of the brush (0.0 - 1.0)]`
- `bool bEnablePressureSensitivity [When set, stylus pressure is used to adjust radius]`
- `bool bHitBackFaces [Allow the Brush to hit the back-side of the mesh]`
- `bool bSpecifyRadius [If true, ignore relative Brush Size and use explicit world Radius]`

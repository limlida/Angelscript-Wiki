### UMirrorModifier


Animation Modifier to mirror an animation using a mirror data table

**属性**:

- `UMirrorDataTable MirrorDataTable [MirrorDataTable that will be used for mirroring the aniamtion]`
- `bool bUpdateNotifies [If notifies should be mirrored in the animation]`
- `bool bUpdateSyncMarkers [If sync markers should be mirrored in the animation]`

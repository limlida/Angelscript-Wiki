### UGizmoBoxComponent


Simple Component intended to be used as part of 3D Gizmos.
Draws outline of 3D Box based on parameters.

**属性**:

- `FVector Dimensions []`
- `float32 LineThickness []`
- `FVector Origin []`
- `FQuat Rotation []`
- `bool bEnableAxisFlip []`
- `bool bRemoveHiddenLines []`

### UAnimBoneCompressionCodec_ACLBase


The base codec implementation for ACL support.

**属性**:

- `ACLCompressionLevel CompressionLevel [The compression level to use. Higher levels will be slower to compress but yield a lower memory footprint.]`
- `float32 DefaultVirtualVertexDistance [The default virtual vertex distance for normal bones.]`
- `float32 ErrorThreshold [The error threshold to use when optimizing and compressing the animation sequence.]`
- `ACLPhantomTrackMode PhantomTrackMode [How to treat phantom tracks. Phantom tracks are not mapped to a skeleton bone.]`
- `float32 SafeVirtualVertexDistance [The virtual vertex distance for bones that requires extra accuracy.]`

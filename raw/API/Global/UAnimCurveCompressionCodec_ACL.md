### UAnimCurveCompressionCodec_ACL


Uses the open source Animation Compression Library with default settings suitable for general purpose animation curves.

**属性**:

- `float32 CurvePrecision [The curve precision to target when compressing the animation curves.]`
- `float32 MorphTargetPositionPrecision [The mesh deformation precision to target when compressing morph target animation curves.]`
- `USkeletalMesh MorphTargetSource [The skeletal mesh used to estimate the morph target deformation during compression.]`

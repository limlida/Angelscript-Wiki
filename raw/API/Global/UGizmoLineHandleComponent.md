### UGizmoLineHandleComponent


Simple Component intended to be used as part of 3D Gizmos.
Draws line terminated by a short perpendicular handle line based on parameters.

**属性**:

- `FVector Direction []`
- `float32 HandleSize []`
- `float32 Length []`
- `FVector Normal []`
- `float32 Thickness []`
- `bool bImageScale []`

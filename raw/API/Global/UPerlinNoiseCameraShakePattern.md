### UPerlinNoiseCameraShakePattern


A camera shake that uses Perlin noise to shake the camera.

**属性**:

- `FPerlinNoiseShaker FOV [FOV shake.]`
- `float32 LocationAmplitudeMultiplier [Amplitude multiplier for all location shake]`
- `float32 LocationFrequencyMultiplier [Frequency multiplier for all location shake]`
- `FPerlinNoiseShaker Pitch [Pitch shake.]`
- `FPerlinNoiseShaker Roll [Roll shake.]`
- `float32 RotationAmplitudeMultiplier [Amplitude multiplier for all rotation shake]`
- `float32 RotationFrequencyMultiplier [Frequency multiplier for all rotation shake]`
- `FPerlinNoiseShaker X [Shake in the X axis.]`
- `FPerlinNoiseShaker Y [Shake in the Y axis.]`
- `FPerlinNoiseShaker Yaw [Yaw shake.]`
- `FPerlinNoiseShaker Z [Shake in the Z axis.]`

### UChaosVDJointConstraintsVisualizationSettings


**属性**:

- `float32 AngularImpulseScale [Scale to apply to the Angular Impulse vector before draw it.]`
- `float32 BaseLineThickness [Line thickness to use as a base to calculate the different line thickness values used to debug draw the data.]`
- `float32 CenterOfMassSize [Size of the debug drawn Center Of Mass.]`
- `float32 ConstraintAxisLength [Size of the debug drawn if the Constraint Axis]`
- `ESceneDepthPriorityGroup DepthPriority [The depth priority used for while drawing data. Can be World or Foreground (with this one the shapes will be drawn on top of the geometry and be always visible)]`
- `float32 GeneralScale [Scale to apply to anything that does not have a dedicated scale setting before draw it.]`
- `float32 LinearImpulseScale [Scale to apply to the Linear Impulse vector before draw it.]`
- `bool bShowDebugText [If true, any debug draw text available will be drawn]`

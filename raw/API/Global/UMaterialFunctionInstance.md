### UMaterialFunctionInstance


A material function instance defines parameter overrides for a parent material function.

**属性**:

- `TArray<FDoubleVectorParameterValue> DoubleVectorParameterValues [DoubleVector parameters.]`
- `TArray<FFontParameterValue> FontParameterValues [Font parameters.]`
- `TArray<FParameterCollectionParameterValue> ParameterCollectionParameterValues [Parameter Collection parameters.]`
- `UMaterialFunctionInterface Parent [Parent function.]`
- `TArray<FRuntimeVirtualTextureParameterValue> RuntimeVirtualTextureParameterValues [Runtime virtual texture parameters.]`
- `TArray<FScalarParameterValue> ScalarParameterValues [Scalar parameters.]`
- `TArray<FSparseVolumeTextureParameterValue> SparseVolumeTextureParameterValues [Sparse volume texture parameters.]`
- `TArray<FTextureCollectionParameterValue> TextureCollectionParameterValues [Texture Collection parameters.]`
- `TArray<FTextureParameterValue> TextureParameterValues [Texture parameters.]`
- `TArray<FVectorParameterValue> VectorParameterValues [Vector parameters.]`

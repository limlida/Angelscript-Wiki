### UNavigationToolSettings


**属性**:

- `ENavigationToolItemViewMode ItemDefaultViewMode [The View Mode a Non-Actor / Non-Component Item supports by default]`
- `ENavigationToolItemViewMode ItemProxyViewMode [The View Mode a Proxy Item supports by default]`
- `EFilterBarLayout LastFilterBarLayout [Last saved layout orientation of the filter bar to restore after closed]`
- `bool bAlwaysShowLockState [Whether to show the lock state always, rather than only showing when the item is locked or hovered]`
- `bool bApplyDefaultColumnView [Applies the default column view set by the provider when a view is loaded]`
- `bool bAutoExpandNodesOnFilterPass [Automatically expand tracks that pass filters]`
- `bool bAutoExpandToSelection [Whether to auto expand the hierarchy to show the item when selected]`
- `bool bFilterBarVisible [Last saved visibility of the filter bar to restore after closed]`
- `bool bSyncSelectionToNavigationTool [If true, syncs selections in Sequencer to Navigation Tool]`
- `bool bSyncSelectionToSequencer [If true, syncs selections in Navigation Tool to Sequencer]`
- `bool bUseFilterSubmenusForCategories [Display the filter menu categories as submenus instead of sections]`
- `bool bUseMutedHierarchy [Whether to show the parent of the shown items, even if the parents are filtered out]`
- `bool bUseShortNames [Shortens child item names that contain their parents name as a prefix by excluding it from the displayed child name]`

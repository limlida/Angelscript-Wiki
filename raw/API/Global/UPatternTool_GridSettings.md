### UPatternTool_GridSettings


Settings for Grid Patterns in the Pattern Tool
TODO: maybe we can just re-use UPatternTool_LinearSettings for this??

**属性**:

- `int CountX [Number of Pattern Elements to place along the Main axis]`
- `int CountY [Number of  Pattern Elements to place along the Secondary axis]`
- `float ExtentX [Length/Extent of Pattern falong the Main Axis]`
- `float ExtentY [Length/Extent of Pattern falong the Secondary Axis]`
- `EPatternToolAxisSpacingMode SpacingX [Spacing Technique used to distribute Pattern Elements along the Main axis]`
- `EPatternToolAxisSpacingMode SpacingY [Spacing Technique used to distribute Pattern Elements along the Secondary axis]`
- `float StepSizeX [Fixed Increment used to place Pattern Elements along the Main axis]`
- `float StepSizeY [Fixed Increment used to place Pattern Elements along the Secondary axis]`
- `bool bCenteredX [If true, Pattern is centered at the Origin along the Main axis, otherwise Pattern starts at the Origin]`
- `bool bCenteredY [If true, Pattern is centered at the Origin along the Secondary axis, otherwise Pattern starts at the Origin]`

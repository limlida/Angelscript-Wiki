### UImportVertexColorOptions

- `int LODIndex [LOD Index to import the Vertex Colors to]`
- `int UVIndex [Texture Coordinate Channel to use for Sampling the Texture]`

- `bool bBlue [Blue Texture Channel]`
- `bool bGreen [Green Texture Channel]`
- `bool bImportToInstance [Whether or not to import the Vertex Colors to Mesh Component instance or the underlying Static Mesh]`

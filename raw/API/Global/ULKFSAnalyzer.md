### ULKFSAnalyzer


ULKFSAnalyzer

ULKFSAnalyzer calculates the current amplitude of an
audio bus in real-time.

**属性**:

- `FOnLatestOverallLKFSResults OnLatestOverallLKFSResults [Delegate to receive the latest overall meter results.]`
- `FOnLatestPerChannelLKFSResults OnLatestPerChannelLKFSResults [Delegate to receive the latest per-channel meter results.]`
- `FOnOverallLKFSResults OnOverallLKFSResults [Delegate to receive all overall meter results since last delegate call.]`
- `FOnPerChannelLKFSResults OnPerChannelLKFSResults [Delegate to receive all meter results, per-channel, since last delegate call.]`
- `ULKFSSettings Settings [The settings for the meter audio analyzer.]`

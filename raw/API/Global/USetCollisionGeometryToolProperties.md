### USetCollisionGeometryToolProperties


**属性**:

- `float32 AddHullsErrorTolerance [Error tolerance for adding more convex hulls, in cm.  For volumetric errors, the value will be cubed (so a value of 10 indicates a 10x10x10 volume worth of error is acceptable).]`
- `float32 ConvexDecompositionSearchFactor [How much to search the space of possible decompositions beyond Max Hulls Per Shape; for larger values, will do additional work to try to better approximate mesh features (but resulting hulls may overlap more)]`
- `EConvexDecompositionMethod DecompositionMethod [What algorithm should be used to perform convex decomposition]`
- `float DecompositionTargetEdgeLength [Simplify the input to this edge length before computing convex decomposition. Can give a decomposition result faster for large meshes.]`
- `ECollisionGeometryType GeometryType [What kind of shapes to fit to the input. Note: Will be overridden by any enabled 'Auto Detect' settings, if close-fitting 'Auto Detect' shapes are found.]`
- `int HullTargetFaceCount [Target number of faces in the simplified hull]`
- `float32 HullTolerance [If > 0, the polygon used to generate the swept hull will be simplified up to this distance tolerance, in cm]`
- `ESetCollisionGeometryInputMode InputMode [What parts of the input should be separately fit with collision shapes]`
- `int LevelSetResolution [Level set grid resolution along longest grid axis]`
- `int MaxCount [The maximum number of collision shapes to generate. If necessary, the shapes with smallest volume will be discarded to meet this count.]`
- `int MaxHullsPerShape [Maximum number of convex hulls to use in each decomposition]`
- `int MergeAboveCount [Attempt to merge generated collision shapes until there are at most this many]`
- `float32 MinPartThickness [Minimum part thickness for convex decomposition, in cm; hulls thinner than this will be merged into adjacent hulls, if possible.]`
- `float32 MinThickness [Generated collision shapes will be expanded if they are smaller than this in any dimension. Not supported for Level Sets or Convex Decompositions.]`
- `float NegativeSpaceMinRadius [Minimum radius of navigable space to protect; tunnels with radius smaller than this could be filled in]`
- `float NegativeSpaceTolerance [Navigable space closer to the input than this tolerance distance can be filled in]`
- `ECollisionGeometryMode SetCollisionType [Set how the physics system should interpret collision shapes on the output mesh. Does not affect what collision shapes are generated by this tool.]`
- `EProjectedHullAxis SweepAxis [How to choose which direction to sweep when creating a swept hull]`
- `bool bAppendToExisting [Whether to keep the existing collision shapes, and append new shapes to that set. Otherwise, existing collision will be cleared.]`
- `bool bDetectBoxes [Whether to override the requested Geometry Type with a box whenever a box closely fits the input shape]`
- `bool bDetectCapsules [Whether to override the requested Geometry Type with a capsule whenever a capsule closely fits the input shape]`
- `bool bDetectSpheres [Whether to override the requested Geometry Type with a sphere whenever a sphere closely fits the input shape]`
- `bool bEnableMaxCount [Whether to discard all but MaxCount collision geometries with the largest volume]`
- `bool bIgnoreInternalNegativeSpace [Whether to ignore navigable space that is not accessible by from outside (e.g., closed-off interiors / air pockets)]`
- `bool bLimitHullsPerShape [Whether to limit the number of convex hulls use in each decomposition]`
- `bool bMergeCollisionShapes [Whether to attempt to merge the generated collision shapes, when there are more than MergeAboveCount]`
- `bool bPreSimplifyToEdgeLength [Whether to simplify the input to this edge length before computing convex decomposition. Can give a decomposition result faster for large meshes.]`
- `bool bRemoveContained [Whether to attempt to detect and remove collision shapes that are fully contained inside other collision shapes]`
- `bool bShowTargetMesh [Show/Hide target mesh]`
- `bool bSimplifyHulls [Whether to simplify the convex hull down to at most a target face count.]`
- `bool bUseNegativeSpaceInMerge [Whether to protect negative space while merging the generated collision shapes, using the negative space settings]`
- `bool bUseWorldSpace [When using multiple inputs to generate our collision, whether to use the world-space position of those input. If false, inputs will be considered as if they were all centered at the same location.]`

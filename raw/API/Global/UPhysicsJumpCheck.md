### UPhysicsJumpCheck


WARNING - This class will be removed. Please use UChaosCharacterJumpCheck instead

Transition that handles jumping based on input for a physics-based character

**属性**:

- `float32 FractionalGroundReactionImpulse [Controls how much of the jump impulse the character will apply to the ground.
A value of 0 means no impulse will be applied to the ground.
A value of 1 means that the full equal and opposite jump impulse will be applied.]`
- `float32 JumpUpwardsSpeed [Instantaneous speed induced in an actor upon jumping]`
- `FName TransitionToMode [Name of movement mode to transition to when jumping is activated]`

### UDefaultLevelSequenceInstanceData


Default instance data class that level sequences understand. Implements IMovieSceneTransformOrigin.

**属性**:

- `FTransform TransformOrigin [Specifies a transform that offsets all absolute transform sections in this sequence. Scale is ignored. Not applied to Relative or Additive sections.]`
- `AActor TransformOriginActor [When set, this actor's world position will be used as the transform origin for all absolute transform sections]`

### UEditorProjectAppearanceSettings


Editor project appearance settings. Stored in default config, per-project

**属性**:

- `EUnit AccelerationUnits [Choose the units in which to display acceleration.]`
- `EUnit AngleUnits [Choose the units in which to display angles.]`
- `EUnit AngularSpeedUnits [Choose the units in which to display angular speeds.]`
- `TArray<EUnit> DistanceUnits [Choose a set of units in which to display distance/length values.]`
- `EUnit ForceUnits [Choose the units in which to display forces.]`
- `EUnit ImpulseUnits [Choose the units in which to display impulses.]`
- `TArray<EUnit> MassUnits [Choose a set of units in which to display masses.]`
- `EUnit PositionalImpulseUnits [Choose the units in which to display positional impulses.]`
- `int ReferenceViewerDefaultMaxSearchBreadth [The default maximum search breadth for the reference viewer when opened]`
- `EReferenceViewerSettingMode ShowSearchableNames [Should the Reference Viewer have 'Show Searchable Names' checked by default when opened in this project]`
- `EUnit SpeedUnits [Choose the units in which to display speeds and velocities.]`
- `EUnit TemperatureUnits [Choose the units in which to display temperatures.]`
- `TArray<EUnit> TimeUnits [Choose the units in which to display time.]`
- `EUnit TorqueUnits [Choose the units in which to display torques.]`
- `bool bDisplayUnits [Whether to display units on editor properties where the property has units set.]`
- `bool bDisplayUnitsOnComponentTransforms [Whether to display units on component transform properties]`

### UEditorProjectAssetSettings


Settings for how developers interact with assets. Stored in default config, per-project

**属性**:

- `bool bPromptToDeleteUnreferencedRedirectors [When performing cleanup operations on redirectors (such as resaving their referencers),
prompt the user to delete unreferenced redirectors.]`
- `bool bRenameLocalizedVariantsAlongsideSourceAsset [If the project uses Localization, localized variants of assets will be moved/renamed/deleted alongside their source asset.]`

### UControlRigEditModeSettings


Settings object used to show useful information in the details panel

**属性**:

- `float32 AxisScale [The scale for axes to draw on the selection]`
- `FLinearColor DrivenControlColor [Indicates a control being driven by a proxy control]`
- `float32 GizmoScale [The scale for Gizmos]`
- `FLinearColor TweenOutOfFocusTint [The tint to apply when the tween overlay is out of focus.]`
- `bool bCoordSystemPerWidgetMode [If true we restore the coordinate space when changing Widget Modes in the Viewport]`
- `bool bDisplayAxesOnSelection [Should we show axes for the selected elements]`
- `bool bDisplayHierarchy [Whether to show all bones in the hierarchy]`
- `bool bDisplayNulls [Whether to show all nulls in the hierarchy]`
- `bool bDisplaySockets [Should we show sockets in the viewport]`
- `bool bHideControlShapes [Should we always hide control shapes in viewport]`
- `bool bIndirectSliderMovementShouldSnapSliderToMouse [If enabled, the DragAnimSliderTool hotkey should position the widget at the mouse cursor if the widget was visible when you pressed it.
If disabled, the DragAnimSliderTool hotkey will leave the widget where you placed it if the widget was visible when you pressed it.]`
- `bool bLocalTransformsInEachLocalSpace [If true when we transform multiple selected objects in the viewport they each transforms along their own local transform space]`
- `bool bOnlySelectRigControls [If true we can only select Rig Controls in the scene not other Actors.]`
- `bool bShowAllProxyControls [Should we always hide control shapes in viewport]`
- `bool bShowControlsAsOverlay [Determins if controls should be rendered on top of other controls]`

### UAnimBlueprintFactory


**属性**:

- `EBlueprintType BlueprintType [The type of blueprint that will be created]`
- `TSubclassOf<UAnimInstance> ParentClass [The parent class of the created blueprint]`
- `USkeletalMesh PreviewSkeletalMesh [The preview mesh to use with this animation blueprint]`
- `USkeleton TargetSkeleton [The kind of skeleton that animation graphs compiled from the blueprint will animate]`
- `bool bTemplate [Whether the created blueprint should be a template with no target skeleton]`

### UControlRigNumericalValidationPass


Used to perform a numerical comparison of the poses

**属性**:

- `float32 CurvePrecision [The threshold under which we'll ignore a precision issue in the pass]`
- `float32 RotationPrecision [The threshold under which we'll ignore a precision issue in the pass (in degrees)]`
- `float32 ScalePrecision [The threshold under which we'll ignore a precision issue in the pass]`
- `float32 TranslationPrecision [The threshold under which we'll ignore a precision issue in the pass]`
- `bool bCheckBones [If set to true the pass will validate the poses of all bones]`
- `bool bCheckControls [If set to true the pass will validate the poses of all bones]`
- `bool bCheckCurves [If set to true the pass will validate the values of all curves]`

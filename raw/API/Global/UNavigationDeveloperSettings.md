### UNavigationDeveloperSettings


**属性**:

- `FLinearColor FriendlyColor [If we're friendly the nav marker will appear this color]`
- `FLinearColor HostileColor [If we're hostile the nav marker will appear this color]`
- `FLinearColor NeutralColor [If we're neutral the nav marker will appear this color]`
- `FLinearColor PlayerColor [If we're the local player the nav marker will appear this color]`

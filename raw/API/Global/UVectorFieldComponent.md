### UVectorFieldComponent


A Component referencing a vector field.

**属性**:

- `float32 Intensity [The intensity at which the vector field is applied.]`
- `float32 Tightness [How tightly particles follow the vector field.]`
- `UVectorField VectorField [The vector field asset.]`


**方法**:

- `SetIntensity(float32 NewIntensity)`  
  Set the intensity of the vector field.
@param NewIntensity - The new intensity of the vector field.

### UTkAbilitySet


UTkAbilitySet

    Non-mutable data asset used to grant gameplay abilities and gameplay effects.

**属性**:

- `TArray<FTkAbilitySet_AttributeSet> GrantedAttributes [Attribute sets to grant when this ability set is granted.]`
- `TArray<FTkAbilitySet_GameplayAbility> GrantedGameplayAbilities [Gameplay abilities to grant when this ability set is granted.]`
- `TArray<FTkAbilitySet_GameplayEffect> GrantedGameplayEffects [Gameplay effects to grant when this ability set is granted.]`


**方法**:

- `GiveToAbilitySystem(UTkAbilitySystemComponent TkASC) const`

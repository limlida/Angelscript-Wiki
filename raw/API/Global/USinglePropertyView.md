### USinglePropertyView


The single property view allows you to display the value of an object's property.

**属性**:

- `FText NameOverride [Override for the property name that will be displayed instead of the property name.]`
- `FName PropertyName [The name of the property to display.]`


**方法**:

- `FText GetNameOverride() const`
- `FName GetPropertyName() const`
- `SetNameOverride(FText NewPropertyName)`
- `SetPropertyName(FName NewPropertyName)`

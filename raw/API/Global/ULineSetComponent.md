### ULineSetComponent


**方法**:

- `int AddLines(TArray<FVector> InStart, TArray<FVector> InEnd, FColor InColor, float32 InThickness = 1.000000, float32 InDepthBias = 0.000000)`  
  Add lines to be rendered using the component.
- `Clear()`  
  Clear the line set
- `SetLineMaterial(UMaterialInterface InLineMaterial)`  
  Specify material which handles lines

### USculptMaxBrushOpProps


**属性**:

- `float32 Falloff [Amount of falloff to apply]`
- `float32 FixedHeight [Maximum height in world-space dimension]`
- `float32 MaxHeight [Maximum height as fraction of brush size]`
- `float32 Strength [Strength of the Brush]`
- `bool bUseFixedHeight [If true, maximum height is defined using the FixedHeight constant instead of brush-relative size]`

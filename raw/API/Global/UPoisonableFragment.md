### UPoisonableFragment


Add this to any item that needs to be poisonable - usually weapons, could be ammo like arrows, etc.

**属性**:

- `TSubclassOf<UGameplayEffect> AppliedPoison [The poison that has been applied to this.]`


**方法**:

- `SetPoison(TSubclassOf<UGameplayEffect> Poison)`  
  Set the item as being poisoned

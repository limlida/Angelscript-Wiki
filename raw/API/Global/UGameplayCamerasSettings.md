### UGameplayCamerasSettings


The settings for the Gameplay Cameras runtime.

**属性**:

- `int CombinedCameraRigNumThreshold [The number of camera rigs combined in one frame past which the camera system emits a warning.]`
- `float DefaultIKAimingAngleTolerance [The default angle tolerance to accept an aiming operation.]`
- `float DefaultIKAimingDistanceTolerance [The default distance tolerance to accept an aiming operation.]`
- `uint8 DefaultIKAimingMaxIterations [The default number of iterations for an aiming operation.]`
- `float DefaultIKAimingMinDistance [The distance below which any IK aiming operation is disabled.]`
- `EGameplayCamerasViewRotationMode DefaultViewRotationMode [The default view rotation handling mode to use when the game's player controller uses a
GameplayCamerasPlayerCameraManager instance as its camera manager.]`
- `bool bAutoBuildInPIE [Build camera assets before using them in PIE, so that they automatically run with the
latest user edits.]`

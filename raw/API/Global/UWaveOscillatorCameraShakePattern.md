### UWaveOscillatorCameraShakePattern


A camera shake that uses oscillations to move the camera.

**属性**:

- `FWaveOscillator FOV [FOV oscillation.]`
- `float32 LocationAmplitudeMultiplier [Amplitude multiplier for all location oscillation]`
- `float32 LocationFrequencyMultiplier [Frequency multiplier for all location oscillation]`
- `FWaveOscillator Pitch [Pitch oscillation.]`
- `FWaveOscillator Roll [Roll oscillation.]`
- `float32 RotationAmplitudeMultiplier [Amplitude multiplier for all rotation oscillation]`
- `float32 RotationFrequencyMultiplier [Frequency multiplier for all rotation oscillation]`
- `FWaveOscillator X [Oscillation in the X axis.]`
- `FWaveOscillator Y [Oscillation in the Y axis.]`
- `FWaveOscillator Yaw [Yaw oscillation.]`
- `FWaveOscillator Z [Oscillation in the Z axis.]`

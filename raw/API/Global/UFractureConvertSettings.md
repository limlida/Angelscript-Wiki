### UFractureConvertSettings


Settings related to geometry collection -> static mesh conversion *

**属性**:

- `bool bCenterPivots [Whether to center the pivot for each mesh, or use the pivot from the geometry collection]`
- `bool bPerBone [Whether to generate a separate mesh for each bone, or one combined mesh]`
- `bool bPlaceInWorld [Whether to place new static mesh actors in the level]`
- `bool bPromptForBaseName [Whether to prompt user for a location and base name for the generated meshes, or automatically place them next to the source geometry collections]`
- `bool bSelectNewActors [Whether to select new static mesh actors]`

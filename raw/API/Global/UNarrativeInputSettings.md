- `float32 GetAimSensitivity() const`
- `bool GetInvertHorizontal() const`
- `bool GetInvertVertical() const`
- `SetAimSensitivity(float32 NewAimSensitivity)`
- `SetInvertHorizontal(bool NewInvertHorizontal)`
- `SetInvertVertical(bool NewInvertVertical)`

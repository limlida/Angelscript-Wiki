### UWorldPartitionEditorSettings


**属性**:

- `TSubclassOf<UWorldPartitionConvertCommandlet> CommandletClass [Commandlet class to use for World Partition conversion]`
- `float HLODMaxDrawDistance [Maximum distance at which HLODs should be displayed in editor]`
- `float HLODMinDrawDistance [Minimum distance at which HLODs should be displayed in editor]`
- `int InstancedFoliageGridSize [Editor grid size used for instance foliage actors in World Partition worlds]`
- `int MinimapLowQualityWorldUnitsPerPixelThreshold [Threshold from which minimap generates a warning if its WorldUnitsPerPixel is above this value]`
- `bool bEnableLoadingInEditor [Whether to enable dynamic loading in the editor through loading regions]`
- `bool bEnableStreamingGenerationLogOnPIE [Whether to enable streaming generation log on PIE]`
- `bool bShowHLODsInEditor [Whether to show HLODs in the editor]`
- `bool bShowHLODsOverLoadedRegions [Control display of HLODs in case actors are loaded]`

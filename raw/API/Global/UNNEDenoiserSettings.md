### UNNEDenoiserSettings


Settings to select a NNE Denoiser and its runtime

**属性**:

- `TSoftObjectPtr<UNNEDenoiserAsset> DenoiserAsset [Select the denoiser asset]`
- `int MaximumTileSizeOverride [Override the maximum tile size given by the asset.
Special values:
   -1 = Do not override]`
- `FString RuntimeName [(Optional) Specify the Runtime name]`
- `EDenoiserRuntimeType RuntimeType [Select a Runtime type]`

### UInputTriggerHold


UInputTriggerHold
      Trigger fires once input has remained actuated for HoldTimeThreshold seconds.
      Trigger may optionally fire once, or repeatedly fire.

**属性**:

- `float32 HoldTimeThreshold [How long does the input have to be held to cause trigger?]`
- `bool bIsOneShot [Should this trigger fire only once, or fire every frame once the hold time threshold is met?]`

### UNiagaraParameterDefinitions


Collection of Niagara parameters and their metadata, which is synced to all other assets in the project.
This can be used to enforce consistent naming and default values for commonly used parameters.

**属性**:

- `int MenuSortOrder [Defines the sort order in add menus. Entries with smaller numbers are displayed first.]`
- `bool bPromoteToTopInAddMenus [If true then these parameters will appear as top level entry in add menus (e.g. in the module editor)]`

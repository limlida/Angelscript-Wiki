### UBakeHeightMapToolProperties


**属性**:

- `EBakeHeightRangeMode HeightRangeMode [The coordinate space to compute height]`
- `float32 InnerBoundsDistance [The inner height map distance from the low poly mesh, as a ratio of the maximum bounding box axis. This value maps to black.]`
- `float32 InnerDistance [The inner height map distance from the low poly mesh, in object space units. This value maps to black.]`
- `float32 OuterBoundsDistance [The outer height map distance from the low poly mesh, as a ratio of the maximum bounding box axis. This value maps to white.]`
- `float32 OuterDistance [The outer height map distance from the low poly mesh, in object space units. This value maps to white.]`

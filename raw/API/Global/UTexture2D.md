### UTexture2D


**属性**:

- `TextureAddress AddressX [The addressing mode to use for the X axis.]`
- `TextureAddress AddressY [The addressing mode to use for the Y axis.]`


**方法**:

- `FSharedImageConstRefBlueprint Blueprint_GetCPUCopy() const`
- `int Blueprint_GetSizeX() const`  
  Gets the X size of the texture, in pixels
- `int Blueprint_GetSizeY() const`  
  Gets the Y size of the texture, in pixels

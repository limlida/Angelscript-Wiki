### UAccelerationDecelerationValueInterpolator


A value interpolator that first accelerates up to a maximum speed, and then decelerates
before reaching the target value.

**属性**:

- `float32 Acceleration [The acceleration rate at the start of interpolation.]`
- `float32 Deceleration [The deceleration rate at the end of interpolation.]`
- `float32 MaxSpeed [The maximum speed reachable during interpolation.]`

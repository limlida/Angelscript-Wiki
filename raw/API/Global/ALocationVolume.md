### ALocationVolume


A volume representing a location in the world. Used for World Partition loading regions.

**属性**:

- `FColor DebugColor []`


**方法**:

- `bool IsLoaded() const`  
  Return if this location volume is loaded
- `Load()`  
  Load this location volume
- `Unload()`  
  Unload this location volume

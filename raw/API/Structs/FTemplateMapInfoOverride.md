### FTemplateMapInfoOverride


Used by new level dialog.

**属性**:

- `FText DisplayName [The display name of the map template in the level dialog. If this is empty the thumbnail name will be used]`
- `FSoftObjectPath Map [The path to the template map]`
- `FSoftObjectPath Thumbnail [The thumbnail to display in the new level dialog]`


**方法**:

- `FTemplateMapInfoOverride& opAssign(FTemplateMapInfoOverride Other)`

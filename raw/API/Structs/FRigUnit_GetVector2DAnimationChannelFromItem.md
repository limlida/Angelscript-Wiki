### FRigUnit_GetVector2DAnimationChannelFromItem


Get Vector2D Channel is used to retrieve a control's animation channel value

**属性**:

- `FRigElementKey Item [The item representing the channel]`
- `FVector2D Value [The current value of the animation channel]`
- `bool bInitial [If set to true the initial value will be returned]`


**方法**:

- `FRigUnit_GetVector2DAnimationChannelFromItem& opAssign(FRigUnit_GetVector2DAnimationChannelFromItem Other)`

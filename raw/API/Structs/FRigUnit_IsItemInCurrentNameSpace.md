### FRigUnit_IsItemInCurrentNameSpace


Returns true if the given item has been created by this module,
which means that the item's namespace is the module namespace.

**属性**:

- `FRigElementKey Item [* The key of the item to return the namespace for]`
- `bool Result [* True if the item is in this namespace / owned by this module.]`


**方法**:

- `FRigUnit_IsItemInCurrentNameSpace& opAssign(FRigUnit_IsItemInCurrentNameSpace Other)`

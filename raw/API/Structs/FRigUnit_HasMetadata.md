### FRigUnit_HasMetadata


Returns true if a given item in the hierarchy has a specific set of metadata

**属性**:

- `bool Found [True if the item has the metadata]`
- `FRigElementKey Item [The item to check the metadata for]`
- `FName Name [The name of the metadata to check]`
- `ERigMetaDataNameSpace NameSpace [Defines in which namespace the metadata will be looked up]`
- `ERigMetadataType Type [The type of metadata to check for]`


**方法**:

- `FRigUnit_HasMetadata& opAssign(FRigUnit_HasMetadata Other)`

### FMetaSoundQualitySettings


**属性**:

- `FPerPlatformFloat BlockRate [Block Rate (in Hz). NOTE: A Zero value will have no effect and use the Default (100)  *]`
- `FName Name [Name of this quality setting. This will appear in the quality dropdown list.
              The names should be unique but are not guaranteed to be (use guid for unique match) *]`
- `FPerPlatformInt SampleRate [Sample Rate (in Hz). NOTE: A Zero value will have no effect and use the Device Rate. *]`


**方法**:

- `FMetaSoundQualitySettings& opAssign(FMetaSoundQualitySettings Other)`

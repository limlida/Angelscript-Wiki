### FRigVMFunction_VisualLogLocation


Logs location as sphere with given radius - recording for Visual Logs has to be enabled to record this data

**属性**:

- `FName Category [The category of the logged text]`
- `FRigVMDebugDrawSettings DebugDrawSettings`
- `FRigVMExecutePin ExecutePin [* This property is used to chain multiple mutable nodes together]`
- `FVector Location [The location to log]`
- `FLinearColor ObjectColor [The color of the logged object]`
- `float32 Radius [The radius of the sphere to log the location with]`
- `FString Text [The text to log]`


**方法**:

- `FRigVMFunction_VisualLogLocation& opAssign(FRigVMFunction_VisualLogLocation Other)`

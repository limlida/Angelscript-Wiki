### FAbilityTriggerData


Structure that defines how an ability will be triggered by external events

**属性**:

- `EGameplayAbilityTriggerSource TriggerSource [The type of trigger to respond to]`
- `FGameplayTag TriggerTag [The tag to respond to]`


**方法**:

- `FAbilityTriggerData& opAssign(FAbilityTriggerData Other)`

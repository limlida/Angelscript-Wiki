### FFreeMoveParams


Input parameters for controlled free movement function

**属性**:

- `float32 Acceleration`
- `float32 Deceleration`
- `float32 DeltaSeconds`
- `float32 Friction`
- `float32 MaxSpeed`
- `FVector MoveInput`
- `EMoveInputType MoveInputType`
- `FRotator OrientationIntent`
- `FRotator PriorOrientation`
- `FVector PriorVelocity`
- `float32 TurningBoost`
- `float32 TurningRate`
- `FQuat WorldToGravityQuat`
- `bool bUseAccelerationForVelocityMove`


**方法**:

- `FFreeMoveParams& opAssign(FFreeMoveParams Other)`

### FRigUnit_RemoveMetadataTag


Removes a tag from an item

**属性**:

- `FRigVMExecutePin ExecutePin [* This property is used to chain multiple mutable units together]`
- `FRigElementKey Item [The item to set the metadata for]`
- `ERigMetaDataNameSpace NameSpace [Defines in which namespace the metadata will be looked up]`
- `bool Removed [Returns true if the removal was successful]`
- `FName Tag [The name of the tag to set]`


**方法**:

- `FRigUnit_RemoveMetadataTag& opAssign(FRigUnit_RemoveMetadataTag Other)`

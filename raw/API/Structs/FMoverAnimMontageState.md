### FMoverAnimMontageState


Data about montages that is replicated to simulated clients

**属性**:

- `float32 CurrentPosition [Current position (during playback only)]`
- `UAnimMontage Montage []`
- `float32 PlayRate [Rate at which this montage is intended to play]`
- `float32 StartingMontagePosition [Montage position when started (in unscaled seconds).]`


**方法**:

- `FMoverAnimMontageState& opAssign(FMoverAnimMontageState Other)`

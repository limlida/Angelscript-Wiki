### FMarkerOnPaintData


**属性**:

- `FGameplayTag Domain [the map domain that this marker is displayed in]`
- `FGeometry IconGeometry [paint space geometry of the marker icon widget]`
- `FGeometry MapGeometry [paint space geometry of the map]`
- `FVector2D MapOrigin [map center location in world space]`
- `FVector2D MapPan [current map pan in map space]`
- `FGeometry ParentGeometry [paint space geometry of the root widget]`


**方法**:

- `FMarkerOnPaintData& opAssign(FMarkerOnPaintData Other)`

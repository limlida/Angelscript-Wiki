### FSmartObjectUserCapsuleParams


Struct defining Smart Object user capsule size.

**属性**:

- `float32 Height [Full height of the capsule]`
- `float32 Radius [Radius of the capsule]`
- `float32 StepHeight [Step up height. This space is ignored when testing collisions.]`


**方法**:

- `FSmartObjectUserCapsuleParams& opAssign(FSmartObjectUserCapsuleParams Other)`

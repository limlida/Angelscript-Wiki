### FDialogueContextMapping


**属性**:

- `FDialogueContext Context [The context of the dialogue.]`
- `FString LocalizationKeyFormat [The format string to use when generating the localization key for this context. This must be unique within the owner dialogue wave.
Available format markers:
  * {ContextHash} - A hash generated from the speaker and target voices.]`
- `USoundWave SoundWave [The soundwave to play for this dialogue.]`


**方法**:

- `FDialogueContextMapping& opAssign(FDialogueContextMapping Other)`

### FGameplayTagQueryColumn


A column which filters rows by matching the input against a Gameplay Tag Query.

**属性**:

- `FGameplayTagQuery DefaultRowValue [DefaultRowValue will be assigned to cells when new rows are created]`
- `FInstancedStruct InputValue [The GameplayTagContainer property this column will filter by apply the query on]`
- `bool bDisabled []`


**方法**:

- `FGameplayTagQueryColumn& opAssign(FGameplayTagQueryColumn Other)`

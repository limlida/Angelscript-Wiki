### FCameraRigAssetReference


A structure holding a reference to a camera rig asset, along with the interface parameter
override values.

**属性**:

- `UCameraRigAsset CameraRig [The referenced camera rig.]`
- `FInstancedOverridablePropertyBag Parameters [The camera rig's parameters.]`


**方法**:

- `FCameraRigAssetReference& opAssign(FCameraRigAssetReference Other)`

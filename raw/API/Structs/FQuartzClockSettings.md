### FQuartzClockSettings


UStruct version of settings struct used to initialized a clock

**属性**:

- `FQuartzTimeSignature TimeSignature [Time Signature (defaults to 4/4)]`
- `bool bIgnoreLevelChange [should the clock start Ticking]`


**方法**:

- `FQuartzClockSettings& opAssign(FQuartzClockSettings Other)`

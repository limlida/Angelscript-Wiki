### FNiagaraOutlinerFilterSettings


View settings used in the Niagara Outliner.

**属性**:

- `ENiagaraExecutionState EmitterExecutionState [Only show emitters with the following execution state.]`
- `ENiagaraSimTarget EmitterSimTarget [Only show emitters with this SimTarget.]`
- `ENiagaraExecutionState SystemExecutionState [Only show systems with the following execution state.]`
- `bool bFilterByEmitterExecutionState`
- `bool bFilterByEmitterSimTarget`
- `bool bFilterBySystemCullState`
- `bool bFilterBySystemExecutionState`
- `bool bSystemCullState [Only show system instances with this cull state.]`


**方法**:

- `FNiagaraOutlinerFilterSettings& opAssign(FNiagaraOutlinerFilterSettings Other)`

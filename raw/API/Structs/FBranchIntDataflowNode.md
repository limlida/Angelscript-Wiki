### FBranchIntDataflowNode


Branch between two int inputs based on boolean condition

**属性**:

- `int A [Int input]`
- `int B [Int input]`
- `FLinearColor OverrideColor []`
- `bool bActive [Used to calculate ContainerElementOffsets]`
- `bool bCondition [If true, Output = A, otherwise Output = B]`
- `bool bOverrideColor []`


**方法**:

- `FBranchIntDataflowNode& opAssign(FBranchIntDataflowNode Other)`

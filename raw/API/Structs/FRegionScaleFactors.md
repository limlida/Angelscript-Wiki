### FRegionScaleFactors


**属性**:

- `float32 NegativeHeight [Scale the region in the NEGATIVE height direction. Range is 0-1. Default is 1.0.]`
- `float32 NegativeWidth [Scale the region in the NEGATIVE width direction. Range is 0-1. Default is 1.0.]`
- `float32 PositiveHeight [Scale the region in the POSITIVE height direction. Range is 0-1. Default is 1.0.]`
- `float32 PositiveWidth [Scale the region in the POSITIVE width direction. Range is 0-1. Default is 1.0.]`


**方法**:

- `FRegionScaleFactors& opAssign(FRegionScaleFactors Other)`

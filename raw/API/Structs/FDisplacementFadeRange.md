### FDisplacementFadeRange


**属性**:

- `float32 EndSizePixels [How large the max displacement should be, in on-screen pixels, when fading out should complete, and displacement
should be disabled.
NOTE: This should be a SMALLER number than Start Fade Size.]`
- `float32 StartSizePixels [How large the max displacement should be, in on-screen pixels, when beginning to fade out displacement.
NOTE: This should be a LARGER number than End Fade Size.]`


**方法**:

- `FDisplacementFadeRange& opAssign(FDisplacementFadeRange Other)`

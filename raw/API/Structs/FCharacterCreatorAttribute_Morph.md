### FCharacterCreatorAttribute_Morph


Base class for a morph target that can be changed on the character

**属性**:

- `FName MorphName [The name of the morph]`
- `float32 MorphValue [The value we need to set the morph to]`


**方法**:

- `FCharacterCreatorAttribute_Morph& opAssign(FCharacterCreatorAttribute_Morph Other)`

### FRichCurveKey


One key in a rich, editable float curve

**属性**:

- `ERichCurveInterpMode InterpMode [Interpolation mode between this key and the next]`
- `ERichCurveTangentMode TangentMode [Mode for tangents at this key]`
- `ERichCurveTangentWeightMode TangentWeightMode [If either tangent at this key is 'weighted']`
- `float32 Time [Time at this key]`
- `float32 Value [Value at this key]`


**方法**:

- `FRichCurveKey& opAssign(FRichCurveKey Other)`

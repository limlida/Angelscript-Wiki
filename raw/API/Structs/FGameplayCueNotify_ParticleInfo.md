### FGameplayCueNotify_ParticleInfo


FGameplayCueNotify_ParticleInfo

    Properties that specify how to play a particle effect.

**属性**:

- `UNiagaraSystem NiagaraSystem [Niagara FX system to spawn.]`
- `FGameplayCueNotify_PlacementInfo PlacementInfoOverride [Defines how the particle system will be placed.]`
- `FGameplayCueNotify_SpawnCondition SpawnConditionOverride [Condition to check before spawning the particle system.]`
- `bool bCastShadow [If enabled, this particle system will cast a shadow.]`
- `bool bOverridePlacementInfo [If enabled, use the placement info override and not the default one.]`
- `bool bOverrideSpawnCondition [If enabled, use the spawn condition override and not the default one.]`


**方法**:

- `FGameplayCueNotify_ParticleInfo& opAssign(FGameplayCueNotify_ParticleInfo Other)`

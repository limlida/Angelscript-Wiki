### FRigUnit_GetControlFloat


GetControlFloat is used to retrieve a single Float from a hierarchy.

**属性**:

- `FName Control [The name of the Control to retrieve the Float for.]`
- `float32 FloatValue [The current value of the control.]`
- `float32 Maximum [The maximum value of the control.]`
- `float32 Minimum [The minimum value of the control.]`


**方法**:

- `FRigUnit_GetControlFloat& opAssign(FRigUnit_GetControlFloat Other)`

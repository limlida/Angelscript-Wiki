### FNamedFilmbackPreset


A named bundle of filmback settings used to implement filmback presets

**属性**:

- `FText DisplayName [Display name for the preset.]`
- `FCameraFilmbackSettings FilmbackSettings`
- `FString Name [Name for the preset.]`


**方法**:

- `FNamedFilmbackPreset& opAssign(FNamedFilmbackPreset Other)`

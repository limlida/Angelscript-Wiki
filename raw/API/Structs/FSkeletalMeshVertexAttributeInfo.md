### FSkeletalMeshVertexAttributeInfo


A structure to store user-controllable settings for attributes

**属性**:

- `ESkeletalMeshVertexAttributeDataType DataType [The data type to store the vertex data as for rendering]`
- `FPerPlatformBool EnabledForRender [Whether this vertex attribute should be included in the render data. Requires a rebuild of the render mesh.]`


**方法**:

- `FSkeletalMeshVertexAttributeInfo& opAssign(FSkeletalMeshVertexAttributeInfo Other)`

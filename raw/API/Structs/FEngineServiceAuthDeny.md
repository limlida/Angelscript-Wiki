### FEngineServiceAuthDeny


Implements a message for denying service access to a remote user.

**属性**:

- `FString UserName [Holds the name of the user that denied access.]`
- `FString UserToDeny [Holds the name of the user that access is denied to.]`


**方法**:

- `FEngineServiceAuthDeny& opAssign(FEngineServiceAuthDeny Other)`

### FRigVMFunction_VisualLogCircle


Logs circle - recording for Visual Logs has to be enabled to record this data

**属性**:

- `FName Category [The category of the logged text]`
- `FVector Center [The center of the circle]`
- `FRigVMDebugDrawSettings DebugDrawSettings`
- `FRigVMExecutePin ExecutePin [* This property is used to chain multiple mutable nodes together]`
- `FLinearColor ObjectColor [The color of the logged object]`
- `float32 Radius [The radius of the circle]`
- `FString Text [The text to log]`
- `float32 Thickness [The thickness the circle]`
- `FVector UpAxis [The up axis/normal of the circle's plane]`
- `bool bWireframe [Whether to display as wireframe]`


**方法**:

- `FRigVMFunction_VisualLogCircle& opAssign(FRigVMFunction_VisualLogCircle Other)`

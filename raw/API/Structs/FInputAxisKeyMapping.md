### FInputAxisKeyMapping


Defines a mapping between an axis and key

@see https://docs.unrealengine.com/latest/INT/Gameplay/Input/index.html

**属性**:

- `FName AxisName [Friendly name of axis, e.g "MoveForward"]`
- `FKey Key [Key to bind it to.]`
- `float32 Scale [Multiplier to use for the mapping when accumulating the axis value]`


**方法**:

- `FInputAxisKeyMapping& opAssign(FInputAxisKeyMapping Other)`

### FStateTreeReferenceOverrideItem


Item describing a state tree override for a state with a specific tag.

**属性**:

- `FGameplayTag StateTag [Exact tag used to match against a tag on a linked State Tree state.]`
- `FStateTreeReference StateTreeReference [State Tree and parameters to replace the linked state asset with.]`


**方法**:

- `FStateTreeReferenceOverrideItem& opAssign(FStateTreeReferenceOverrideItem Other)`

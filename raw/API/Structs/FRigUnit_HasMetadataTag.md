### FRigUnit_HasMetadataTag


Returns true if a given item in the hierarchy has a specific tag stored in the metadata

**属性**:

- `bool Found [True if the item has the metadata]`
- `FRigElementKey Item [The item to check the metadata for]`
- `ERigMetaDataNameSpace NameSpace [Defines in which namespace the metadata will be looked up]`
- `FName Tag [The name of the tag to check]`


**方法**:

- `FRigUnit_HasMetadataTag& opAssign(FRigUnit_HasMetadataTag Other)`

### FTilingConfig


Tiling configuration for fixed and dynamic size models

**属性**:

- `int Alignment [Tile size alignment (applies only to dynamic size models)]`
- `int MaxSize [Maximum tile size (applies only to dynamic size models)]`
- `int MinSize [Minimum tile size (applies only to dynamic size models)]`
- `int Overlap [Tile overlap]`


**方法**:

- `FTilingConfig& opAssign(FTilingConfig Other)`

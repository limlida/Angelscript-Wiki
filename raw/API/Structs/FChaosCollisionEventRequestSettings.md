### FChaosCollisionEventRequestSettings


Settings used to define collision event requests

**属性**:

- `float32 MaxDistance [The maximum distance threshold for the results.]`
- `int MaxNumberResults [The maximum number of results to return.]`
- `float32 MinImpulse [The minimum impulse threshold for the results.]`
- `float32 MinMass [The minimum mass threshold for the results (compared with min of particle 1 mass and particle 2 mass).]`
- `float32 MinSpeed [The min speed threshold for the results (compared with min of particle 1 speed and particle 2 speed).]`
- `EChaosCollisionSortMethod SortMethod [The method used to sort the collision events.]`


**方法**:

- `FChaosCollisionEventRequestSettings& opAssign(FChaosCollisionEventRequestSettings Other)`

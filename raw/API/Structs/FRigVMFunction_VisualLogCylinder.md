### FRigVMFunction_VisualLogCylinder


Logs cylinder shape - recording for Visual Logs has to be enabled to record this data

**属性**:

- `FName Category [The category of the logged text]`
- `FRigVMDebugDrawSettings DebugDrawSettings`
- `FVector End [The end of the line segment forming the cylinder]`
- `FRigVMExecutePin ExecutePin [* This property is used to chain multiple mutable nodes together]`
- `FLinearColor ObjectColor [The color of the logged object]`
- `float32 Radius [The radius of the cylinder]`
- `FVector Start [The start of the line segment forming the cylinder]`
- `FString Text [The text to log]`
- `bool bWireframe [Whether to display as wireframe]`


**方法**:

- `FRigVMFunction_VisualLogCylinder& opAssign(FRigVMFunction_VisualLogCylinder Other)`

### FMetasoundFrontendInterfaceUClassOptions


Options used to restrict a corresponding UClass that interface may be applied to.
If unspecified, interface is assumed to be applicable to any arbitrary UClass.

**属性**:

- `FTopLevelAssetPath ClassPath [Path to MetaSound class interface can be added to (ex. UMetaSoundSource or UMetaSound)]`


**方法**:

- `FMetasoundFrontendInterfaceUClassOptions& opAssign(FMetasoundFrontendInterfaceUClassOptions Other)`

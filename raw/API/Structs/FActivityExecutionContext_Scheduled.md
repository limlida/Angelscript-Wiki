### FActivityExecutionContext_Scheduled


Context for an NPCs schedule running the activity

**属性**:

- `float32 StartTime [When the activity starts and ends]`
- `float32 TimeAtStart [When the activity ends]`
- `bool bStartedFromAdvanceTime [Whether the activity was started from advance time]`


**方法**:

- `FActivityExecutionContext_Scheduled& opAssign(FActivityExecutionContext_Scheduled Other)`

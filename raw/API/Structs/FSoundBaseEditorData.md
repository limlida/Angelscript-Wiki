### FSoundBaseEditorData


This struct allows us to modify the transient editor properties and minimize impact on the properties

**属性**:

- `EUseSubmixOnPreviewMode OverrideUseSubmixesOnPreview [Override the Editor Preference to use submixes on preview playback]`


**方法**:

- `FSoundBaseEditorData& opAssign(FSoundBaseEditorData Other)`

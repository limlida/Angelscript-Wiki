### FSkeletalMeshSamplingRegionBoneFilter


Filter to include or exclude bones and their associated triangles from a sampling region.

**属性**:

- `FName BoneName []`
- `bool bApplyToChildren [If true, this filter will apply to all children of this bone as well.]`
- `bool bIncludeOrExclude [If true, this filter will include bones. If false, it will exclude them.]`


**方法**:

- `FSkeletalMeshSamplingRegionBoneFilter& opAssign(FSkeletalMeshSamplingRegionBoneFilter Other)`

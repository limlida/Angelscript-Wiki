### FRigUnit_ToRigSpace_Location


Converts a position / location from world space to rig (global) space

**属性**:

- `FVector Global [The result position / location in global / rig space]`
- `FVector Value [The input position / location in world]`


**方法**:

- `FRigUnit_ToRigSpace_Location& opAssign(FRigUnit_ToRigSpace_Location Other)`

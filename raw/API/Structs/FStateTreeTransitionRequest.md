### FStateTreeTransitionRequest


Transition request

**属性**:

- `EStateTreeSelectionFallback Fallback [Fallback of the transition if it fails to select the target state]`
- `EStateTreeTransitionPriority Priority [Priority of the transition.]`
- `FStateTreeStateHandle TargetState [Target state of the transition.]`


**方法**:

- `FStateTreeTransitionRequest& opAssign(FStateTreeTransitionRequest Other)`

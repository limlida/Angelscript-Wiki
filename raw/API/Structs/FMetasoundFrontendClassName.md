### FMetasoundFrontendClassName


Name of a Metasound class

- `FName Name [Name of class.]`
- `FName Namespace [Namespace of class.]`
- `FName Variant [Variant of class. The Variant is used to describe an equivalent class which performs the same operation but on differing types.]`


**方法**:

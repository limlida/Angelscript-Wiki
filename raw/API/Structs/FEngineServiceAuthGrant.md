### FEngineServiceAuthGrant


Implements a message for granting service access to a remote user.

**属性**:

- `FString UserName [Holds the name of the user that granted access.]`
- `FString UserToGrant [Holds the name of the user that access is granted to.]`


**方法**:

- `FEngineServiceAuthGrant& opAssign(FEngineServiceAuthGrant Other)`

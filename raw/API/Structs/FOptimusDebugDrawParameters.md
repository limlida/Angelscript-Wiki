### FOptimusDebugDrawParameters


User controllable debug draw settings.

**属性**:

- `int FontSize [Font size for characters.]`
- `int MaxCharacterCount [Space to allocate for character collection.]`
- `int MaxLineCount [Space to allocate for line collection.]`
- `int MaxTriangleCount [Space to allocate for triangle collection.]`
- `bool bForceEnable [Force enable debug rendering.
Otherwise "r.ShaderPrint 1" needs to be set.]`


**方法**:

- `FOptimusDebugDrawParameters& opAssign(FOptimusDebugDrawParameters Other)`

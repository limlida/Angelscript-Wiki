### FAudioMaterialButtonStyle


Represents the appearance of an Audio Material Button

**属性**:

- `FLinearColor ButtonAccentColor [The button's Accent color value.]`
- `FLinearColor ButtonMainColor [The button's Main color value.]`
- `FLinearColor ButtonMainColorTint_1 [The button color's Tint value covering one half of the gradient.]`
- `FLinearColor ButtonMainColorTint_2 [The button color's Tint value covering the other half of the gradient.]`
- `FLinearColor ButtonPressedOutlineColor [The button's Outline color value when Pressed.]`
- `FLinearColor ButtonShadowColor [The button's Shadow color value.]`
- `FLinearColor ButtonUnpressedOutlineColor [The button's Outline color value when Unpressed.]`
- `FVector2f DesiredSize [Desired Draw size of the rendered material]`
- `UMaterialInterface Material [Material used to render the Slate]`


**方法**:

- `FAudioMaterialButtonStyle& opAssign(FAudioMaterialButtonStyle Other)`

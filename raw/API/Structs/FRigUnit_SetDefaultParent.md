### FRigUnit_SetDefaultParent


Changes the default parent for an item - this removes all other current parents.

**属性**:

- `FRigElementKey Child [* The child to be parented under the new default parent]`
- `FRigVMExecutePin ExecutePin [* This property is used to chain multiple mutable units together]`
- `FRigElementKey Parent [* The default parent to be used for the child]`


**方法**:

- `FRigUnit_SetDefaultParent& opAssign(FRigUnit_SetDefaultParent Other)`

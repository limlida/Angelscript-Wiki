### FDataflowSimulationAsset


Dataflow simulation asset (should be in the interface children)

**属性**:

- `UDataflow DataflowAsset [Simulation dataflow asset used to advance in time on Pt]`
- `TSet<FString> SimulationGroups [Simulation groups used to filter within the simulation nodes]`


**方法**:

- `FDataflowSimulationAsset& opAssign(FDataflowSimulationAsset Other)`

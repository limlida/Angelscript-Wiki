### FRigUnit_HasMetadataTagArray


Returns true if a given item in the hierarchy has all of the provided tags

**属性**:

- `bool Found [True if the item has the metadata]`
- `FRigElementKey Item [The item to check the metadata for]`
- `ERigMetaDataNameSpace NameSpace [Defines in which namespace the metadata will be looked up]`
- `TArray<FName> Tags [The name of the tag to check]`


**方法**:

- `FRigUnit_HasMetadataTagArray& opAssign(FRigUnit_HasMetadataTagArray Other)`

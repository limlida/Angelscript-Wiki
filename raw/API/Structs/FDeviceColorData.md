### FDeviceColorData


Data required for setting the Input Device Color

**属性**:

- `FColor LightColor [The color to set the light on]`
- `bool bEnable [True if the light should be enabled at all]`
- `bool bResetAfterCompletion [If true, the light color will be reset to "off" after this property has been evaluated.]`


**方法**:

- `FDeviceColorData& opAssign(FDeviceColorData Other)`

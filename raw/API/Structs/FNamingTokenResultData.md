### FNamingTokenResultData


Evaluated results from a template file string.

**属性**:

- `FText EvaluatedText [The full text with evaluated tokens.]`
- `FText OriginalText [Original text without any modifications.]`
- `TArray<FNamingTokenValueData> TokenValues [The result of individual tokens, in the order they appear in OriginalText.]`


**方法**:

- `FNamingTokenResultData& opAssign(FNamingTokenResultData Other)`

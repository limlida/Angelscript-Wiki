### FPyTestChildStruct


Struct to allow testing of inheritance on Python wrapped types.

**属性**:

- `bool Bool`
- `bool BoolDefaultsOnly []`
- `bool BoolInstanceOnly []`
- `bool BoolMutable []`
- `EPyTestEnum Enum`
- `float32 Float`
- `int Int`
- `FName Name`
- `FString String`
- `TArray<FString> StringArray`
- `TMap<FString,int> StringIntMap`
- `TSet<FString> StringSet`
- `FText Text`


**方法**:

- `FPyTestChildStruct& opAssign(FPyTestChildStruct Other)`

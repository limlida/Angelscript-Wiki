### FChaosBreakingEventRequestSettings


Settings used to refine breaking event requests

**属性**:

- `float32 MaxDistance [The maximum distance threshold for the results.]`
- `int MaxNumberOfResults [The maximum number of results to return.]`
- `float32 MinMass [The minimum mass threshold for the results.]`
- `float32 MinRadius [The minimum breaking radius threshold for the results.]`
- `float32 MinSpeed [The minimum speed threshold for the results.]`
- `EChaosBreakingSortMethod SortMethod [The method used to sort the breaking events.]`

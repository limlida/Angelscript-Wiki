### FAITestSpawnSet


FAITestSpawnSet

Generic AI Test Spawn Set that is used in regular AFunctionalAITest tests.

**属性**:

- `AActor FallbackSpawnLocation [location used for spawning if spawn info doesn't define one]`
- `FName Name [give the set a name to help identify it if need be]`
- `TArray<FAITestSpawnInfo> SpawnInfoContainer [what to spawn]`
- `bool bEnabled`


**方法**:

- `FAITestSpawnSet& opAssign(FAITestSpawnSet Other)`

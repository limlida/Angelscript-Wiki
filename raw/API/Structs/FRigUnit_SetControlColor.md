### FRigUnit_SetControlColor


SetControlColor is used to change the control's color

**属性**:

- `FLinearColor Color [The color to set for the control]`
- `FName Control [The name of the Control to set the color for.]`
- `FRigVMExecutePin ExecutePin [* This property is used to chain multiple mutable units together]`


**方法**:

- `FRigUnit_SetControlColor& opAssign(FRigUnit_SetControlColor Other)`

### FRigUnit_IsInteracting


Returns true if the Control Rig is being interacted

**属性**:

- `TArray<FRigElementKey> Items [The items being interacted on]`
- `bool bIsInteracting [True if there is currently an interaction happening]`
- `bool bIsRotating [True if the current interaction is a rotation]`
- `bool bIsScaling [True if the current interaction is scaling]`
- `bool bIsTranslating [True if the current interaction is a translation]`


**方法**:

- `FRigUnit_IsInteracting& opAssign(FRigUnit_IsInteracting Other)`

### FInputChord


An Input Chord is a key and the modifier keys that are to be held with it.

**属性**:

- `FKey Key [The Key is the core of the chord.]`
- `bool bAlt [Whether the alt key is part of the chord.]`
- `bool bCmd [Whether the command key is part of the chord.]`
- `bool bCtrl [Whether the control key is part of the chord.]`
- `bool bShift [Whether the shift key is part of the chord.]`


**方法**:

- `FInputChord& opAssign(FInputChord Other)`

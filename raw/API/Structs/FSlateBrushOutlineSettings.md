### FSlateBrushOutlineSettings


Possible options for rounded box brush image

**属性**:

- `FSlateColor Color [Tinting applied to the border outline.]`
- `FVector4 CornerRadii [Radius in Slate Units applied to the outline at each corner. X = Top Left, Y = Top Right, Z = Bottom Right, W = Bottom Left]`
- `ESlateBrushRoundingType RoundingType [The Rounding Type *]`
- `float32 Width [Line width in Slate Units applied to the border outline.]`
- `bool bUseBrushTransparency [True if we should use the owning brush's transparency as our own *]`


**方法**:

- `FSlateBrushOutlineSettings& opAssign(FSlateBrushOutlineSettings Other)`

### FRigUnit_GetControlVisibility


GetControlVisibility is used to retrieve the visibility of a control

**属性**:

- `FRigElementKey Item [The name of the Control to set the visibility for.]`
- `bool bVisible [The visibility of the control]`


**方法**:

- `FRigUnit_GetControlVisibility& opAssign(FRigUnit_GetControlVisibility Other)`

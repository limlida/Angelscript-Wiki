### FRigUnit_IsItemInCurrentModule


Returns true if the given item has been created by this module.

**属性**:

- `FRigElementKey Item [* The key of the item to check the module for]`
- `bool Result [* True if the item is owned by this module.]`


**方法**:

- `FRigUnit_IsItemInCurrentModule& opAssign(FRigUnit_IsItemInCurrentModule Other)`

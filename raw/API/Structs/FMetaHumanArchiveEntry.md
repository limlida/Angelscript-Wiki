### FMetaHumanArchiveEntry


A single item in a MetaHuman Archive

**属性**:

- `FString Path [The path to this file relative to the root of the archive]`
- `FString Version [The version of the file in format Major.Minor. If no version is available the version should be 0.0]`


**方法**:

- `FMetaHumanArchiveEntry& opAssign(FMetaHumanArchiveEntry Other)`

### FRigVMFunction_VisualLogCapsule


Logs capsule shape - recording for Visual Logs has to be enabled to record this data

**属性**:

- `FVector Base [The base or origin of the capsule]`
- `FName Category [The category of the logged text]`
- `FRigVMDebugDrawSettings DebugDrawSettings`
- `FRigVMExecutePin ExecutePin [* This property is used to chain multiple mutable nodes together]`
- `float32 HalfHeight [Half the height of the capsule]`
- `FLinearColor ObjectColor [The color of the logged object]`
- `float32 Radius [The radius of the capsule]`
- `FQuat Rotation [The orientation of the capsule]`
- `FString Text [The text to log]`
- `bool bWireframe [Whether to display as wireframe]`


**方法**:

- `FRigVMFunction_VisualLogCapsule& opAssign(FRigVMFunction_VisualLogCapsule Other)`

### FRigUnit_HierarchyImportFromSkeleton


Imports all bones (and curves) from the currently assigned skeleton.
Note: This node only runs as part of the construction event.

**属性**:

- `FRigVMExecutePin ExecutePin [* This property is used to chain multiple mutable units together]`
- `TArray<FRigElementKey> Items []`
- `FName NameSpace`
- `bool bIncludeCurves`
- `bool bIncludeMeshSockets`
- `bool bIncludeVirtualBones`


**方法**:

- `FRigUnit_HierarchyImportFromSkeleton& opAssign(FRigUnit_HierarchyImportFromSkeleton Other)`

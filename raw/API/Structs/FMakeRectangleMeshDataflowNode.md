### FMakeRectangleMeshDataflowNode


Make a rectangle mesh

**属性**:

- `float32 Height [Height of rectangle]`
- `int HeightVertexCount [Number of vertices along Height axis]`
- `FVector Normal [Normal vector of all vertices will be set to this value. Default is +Z axis.]`
- `FVector Origin [Rectangle will be translated so that center is at this point]`
- `FLinearColor OverrideColor []`
- `float32 Width [Width of rectangle]`
- `int WidthVertexCount [Number of vertices along Width axis]`
- `bool bActive [Used to calculate ContainerElementOffsets]`
- `bool bOverrideColor []`


**方法**:

- `FMakeRectangleMeshDataflowNode& opAssign(FMakeRectangleMeshDataflowNode Other)`

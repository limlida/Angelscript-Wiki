### FCollectionSelectTransformStringDataflowNode


Selects transforms by name using a the BoneName attributeor other Transform group string typed attributes

**属性**:

- `FString Attribute [Text to serach in the name]`
- `EDataflowCollectionSelectionByNameMethod Method [Method to use to match the name]`
- `FLinearColor OverrideColor []`
- `FString SearchText [Text to serach in the name]`
- `bool bActive [Used to calculate ContainerElementOffsets]`
- `bool bOverrideColor []`


**方法**:

- `FCollectionSelectTransformStringDataflowNode& opAssign(FCollectionSelectTransformStringDataflowNode Other)`

### FRigUnit_ToRigSpace_Rotation


Converts a rotation from world space to rig (global) space

**属性**:

- `FQuat Global [The result rotation in global / rig space]`
- `FQuat Value [The input rotation in world]`


**方法**:

- `FRigUnit_ToRigSpace_Rotation& opAssign(FRigUnit_ToRigSpace_Rotation Other)`

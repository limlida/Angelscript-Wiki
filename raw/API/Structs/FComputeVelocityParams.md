### FComputeVelocityParams


Input parameters for compute velocity function

**属性**:

- `float32 Acceleration`
- `float32 Deceleration`
- `float32 DeltaSeconds`
- `float32 Friction`
- `FVector InitialVelocity`
- `float32 MaxSpeed [AuxState variables]`
- `FVector MoveDirectionIntent`
- `FVector MoveInput`
- `EMoveInputType MoveInputType`
- `float32 TurningBoost`
- `bool bUseAccelerationForVelocityMove`


**方法**:

- `FComputeVelocityParams& opAssign(FComputeVelocityParams Other)`

### FPhysicsControlNamedControlMultiplierParameters


Data that can be used to parameterize (modify/update) a control multiplier

**属性**:

- `FPhysicsControlSparseMultiplier Data`
- `FName Name [The name of the control (or set of controls) to update]`


**方法**:

- `FPhysicsControlNamedControlMultiplierParameters& opAssign(FPhysicsControlNamedControlMultiplierParameters Other)`

### FRigUnit_GetAnimationChannelFromItemBase


Get Animation Channel is used to retrieve a control's animation channel value

**属性**:

- `FRigElementKey Item [The item representing the channel]`
- `bool bInitial [If set to true the initial value will be returned]`


**方法**:

- `FRigUnit_GetAnimationChannelFromItemBase& opAssign(FRigUnit_GetAnimationChannelFromItemBase Other)`

### FObjectClassColumn


A column which filters rows using an Object reference variable, by checking if that object is of a certain Class.

**属性**:

- `FChooserObjectClassRowData DefaultRowValue [DefaultRowValue will be assigned to cells when new rows are created]`
- `FInstancedStruct InputValue [The Object reference property this column will filter based on]`
- `bool bDisabled []`


**方法**:

- `FObjectClassColumn& opAssign(FObjectClassColumn Other)`

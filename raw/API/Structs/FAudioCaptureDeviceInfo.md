### FAudioCaptureDeviceInfo


Struct defining the time synth global quantization settings

**属性**:

- `FName DeviceName [The name of the audio capture device]`
- `int NumInputChannels [The number of input channels]`
- `int SampleRate [The sample rate of the audio capture device]`

### FAnimationGroupReference


**属性**:

- `FName GroupName [The group name that we synchronize with (NAME_None if it is not part of any group).]`
- `EAnimGroupRole GroupRole [The role this animation can assume within the group (ignored if GroupName is not set)]`
- `EAnimSyncMethod Method [How this animation will synchronize with other animations.]`


**方法**:

- `FAnimationGroupReference& opAssign(FAnimationGroupReference Other)`

### FRigVMFunction_VisualLogSegment


Logs segment - recording for Visual Logs has to be enabled to record this data

**属性**:

- `FName Category [The category of the logged text]`
- `FRigVMDebugDrawSettings DebugDrawSettings`
- `FRigVMExecutePin ExecutePin [* This property is used to chain multiple mutable nodes together]`
- `FLinearColor ObjectColor [The color of the logged object]`
- `FVector SegmentEnd [The end of the line segment]`
- `FVector SegmentStart [The start of the line segment]`
- `FString Text [The text to log]`
- `float32 Thickness [The thickness the circle]`


**方法**:

- `FRigVMFunction_VisualLogSegment& opAssign(FRigVMFunction_VisualLogSegment Other)`

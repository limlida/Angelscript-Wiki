### FRigVMFunction_Sequence


Allows for a single execution pulse to trigger a series of events in order.

**属性**:

- `FRigVMExecuteContext A [The execution result A]`
- `FRigVMExecuteContext B [The execution result B]`
- `FRigVMExecuteContext ExecuteContext [The execution input]`


**方法**:

- `FRigVMFunction_Sequence& opAssign(FRigVMFunction_Sequence Other)`

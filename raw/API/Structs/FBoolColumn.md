### FBoolColumn


A column that filters rows based on the value of a Bool.

**属性**:

- `EBoolColumnCellValue DefaultRowValue [DefaultRowValue will be assigned to cells when new rows are created]`
- `FInstancedStruct InputValue [The bool property this column will filter based on]`
- `TArray<EBoolColumnCellValue> RowValuesWithAny []`
- `bool bDisabled []`


**方法**:

- `FBoolColumn& opAssign(FBoolColumn Other)`

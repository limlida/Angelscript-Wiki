### FMediaPlayerOptions


**属性**:

- `EMediaPlayerOptionBooleanOverride Loop [How to initially select looping of the media.]`
- `EMediaPlayerOptionBooleanOverride PlayOnOpen [How to handle automatic playback when media opens.]`
- `FTimespan SeekTime [Initial media time to start playback at.]`
- `EMediaPlayerOptionSeekTimeType SeekTimeType [How to interpret the initial seek time.]`
- `EMediaPlayerOptionTrackSelectMode TrackSelection [How the initial media tracks for playback are selected]`
- `FMediaPlayerTrackOptions Tracks [Indices of the media tracks to select for playback]`
- `FMediaPlayerInitialTrackLanguageSelection TracksByLanguage [Language of the media tracks to select for playback]`


**方法**:

- `FMediaPlayerOptions& opAssign(FMediaPlayerOptions Other)`

### FRigUnit_SetFloatAnimationChannel


Set Float Channel is used to set a control's animation channel value

**属性**:

- `FName Channel [The name of the animation channel to retrieve the value for.]`
- `FName Control [The name of the Control to retrieve the value for.]`
- `FRigVMExecutePin ExecutePin`
- `float32 Value [The new value of the animation channel]`
- `bool bInitial [If set to true the initial value will be returned]`


**方法**:

- `FRigUnit_SetFloatAnimationChannel& opAssign(FRigUnit_SetFloatAnimationChannel Other)`

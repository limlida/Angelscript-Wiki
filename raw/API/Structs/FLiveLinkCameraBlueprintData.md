### FLiveLinkCameraBlueprintData


Facility structure to handle camera data in blueprint

**属性**:

- `FLiveLinkCameraFrameData FrameData [Dynamic data that can change every frame]`
- `FLiveLinkCameraStaticData StaticData [Static data that should not change every frame]`


**方法**:

- `FLiveLinkCameraBlueprintData& opAssign(FLiveLinkCameraBlueprintData Other)`

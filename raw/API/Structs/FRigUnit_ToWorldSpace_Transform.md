### FRigUnit_ToWorldSpace_Transform


Converts a transform from rig (global) space to world space

**属性**:

- `FTransform Value [The input transform in global / rig space]`
- `FTransform World [The result transform in world space]`


**方法**:

- `FRigUnit_ToWorldSpace_Transform& opAssign(FRigUnit_ToWorldSpace_Transform Other)`

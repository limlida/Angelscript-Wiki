### FDistanceDatum


**属性**:

- `float32 FadeInDistanceEnd [The distance at which this sound has faded in completely.]`
- `float32 FadeInDistanceStart [The FadeInDistance at which to start hearing this sound.
       * If you want to hear the sound up close then setting this to 0 might be a good option.]`
- `float32 FadeOutDistanceEnd [The distance at which this sound is no longer audible.]`
- `float32 FadeOutDistanceStart [The distance at which this sound starts fading out.]`
- `float32 Volume [The volume for which this Input should be played.]`


**方法**:

- `FDistanceDatum& opAssign(FDistanceDatum Other)`

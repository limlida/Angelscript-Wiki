### FRigUnit_ToRigSpace_Transform


Converts a transform from world space to rig (global) space

**属性**:

- `FTransform Global [The result transform in global / rig space]`
- `FTransform Value [The input transform in world]`


**方法**:

- `FRigUnit_ToRigSpace_Transform& opAssign(FRigUnit_ToRigSpace_Transform Other)`

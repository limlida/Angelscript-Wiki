### FRigVMFunction_VisualLogText


Logs simple text string with Visual Logger - recording for Visual Logs has to be enabled to record this data

**属性**:

- `FName Category [The category of the logged text]`
- `FRigVMDebugDrawSettings DebugDrawSettings`
- `FRigVMExecutePin ExecutePin [* This property is used to chain multiple mutable nodes together]`
- `FString Text [The text to log]`


**方法**:

- `FRigVMFunction_VisualLogText& opAssign(FRigVMFunction_VisualLogText Other)`

### FRigUnit_GetRotatorAnimationChannelFromItem


Get Rotator Channel is used to retrieve a control's animation channel value

**属性**:

- `FRigElementKey Item [The item representing the channel]`
- `FRotator Value [The current value of the animation channel]`
- `bool bInitial [If set to true the initial value will be returned]`


**方法**:

- `FRigUnit_GetRotatorAnimationChannelFromItem& opAssign(FRigUnit_GetRotatorAnimationChannelFromItem Other)`

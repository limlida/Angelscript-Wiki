### FRigUnit_DebugPose


Draws vectors on each bone in the viewport across the entire pose

**属性**:

- `FLinearColor Color`
- `FRigVMDebugDrawSettings DebugDrawSettings`
- `FRigVMExecutePin ExecutePin`
- `TArray<FRigElementKey> Items [the items to draw the pose cache for.
if this is empty we'll draw the whole pose cache]`
- `FRigPose Pose`
- `float32 Scale`
- `float32 Thickness`
- `FTransform WorldOffset`
- `bool bEnabled`


**方法**:

- `FRigUnit_DebugPose& opAssign(FRigUnit_DebugPose Other)`

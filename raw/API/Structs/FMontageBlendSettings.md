### FMontageBlendSettings


Montage blend settings. Can be used to overwrite default Montage settings on Play/Stop

**属性**:

- `FAlphaBlendArgs Blend [AlphaBlend options (time, curve, etc.)]`
- `EMontageBlendMode BlendMode [Type of blend mode (Standard vs Inertial)]`
- `UBlendProfile BlendProfile [Blend Profile to use for this blend]`


**方法**:

- `FMontageBlendSettings& opAssign(FMontageBlendSettings Other)`

### FQuartzPulseOverrideStep


Allows the user to specify non-uniform beat durations in odd meters

**属性**:

- `int NumberOfPulses [The number of pulses for this beat duration]`
- `EQuartzCommandQuantization PulseDuration [This Beat duration]`


**方法**:

- `FQuartzPulseOverrideStep& opAssign(FQuartzPulseOverrideStep Other)`

### FNiagaraDataChannelReference


A reference to a Niagara Data Channel with a pre-typed access context for convenience.

**属性**:

- `FNDCAccessContextInst AccessContext`
- `UNiagaraDataChannelAsset DataChannel`
- `bool bCustomAccessContext [If true this reference will provide a custom access context.]`


**方法**:

- `FNiagaraDataChannelReference& opAssign(FNiagaraDataChannelReference Other)`

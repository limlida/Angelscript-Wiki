### FCameraShakeAssetReference


A structure holding a reference to a camera shake asset, along with the interface parameter
override values.

**属性**:

- `UCameraShakeAsset CameraShake [The referenced camera shake.]`
- `FInstancedOverridablePropertyBag Parameters [The camera rig's parameters.]`


**方法**:

### FSetAnchorStateDataflowNode

- `EAnchorStateEnum AnchorState [What anchor state to set on selected bones]`
- `FLinearColor OverrideColor []`
- `bool bActive [Used to calculate ContainerElementOffsets]`
- `bool bOverrideColor []`
- `bool bSetNotSelectedBonesToOppositeState [If true, sets the non selected bones to opposite anchor state]`


**方法**:

### FCreatorMeshMaterial


Defines a material we need to apply to the set mesh, along with any params to set on it

**属性**:

- `TSoftObjectPtr<UMaterialInterface> Material [The material to apply]`
- `TArray<FCreatorMeshMaterialParam_Scalar> ScalarParams [The scalar parameter values to apply to the material]`
- `TArray<FCreatorMeshMaterialParam_Vector> VectorParams [The vector parameter values to apply to the material]`


**方法**:

- `FCreatorMeshMaterial& opAssign(FCreatorMeshMaterial Other)`

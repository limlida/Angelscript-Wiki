### FMassLookAtTaskInstanceData


Task to assign a LookAt target for mass processing

**属性**:

- `float32 Duration [Delay before the task ends. Default (0 or any negative) will run indefinitely so it requires a transition in the state tree to stop it.]`
- `FMassEntityHandle TargetEntity [Entity to set as the target for the LookAt behavior.]`


**方法**:

- `FMassLookAtTaskInstanceData& opAssign(FMassLookAtTaskInstanceData Other)`

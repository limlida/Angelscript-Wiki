### FAIESelectionSetItem


Main selection set item, has it's name, it's view data, and array of names making up the set

**属性**:

- `FText ItemName [The Name of the Set]`
- `TArray<FAIESelectionSetItemName> Names [The collection of set items that make up the set, basically an array of names with some metadata]`
- `FAIESelectionSetItemViewData ViewData [Viewing Data for the set, like location and color]`


**方法**:

- `FAIESelectionSetItem& opAssign(FAIESelectionSetItem Other)`

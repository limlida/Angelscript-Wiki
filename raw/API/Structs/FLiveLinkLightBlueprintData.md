### FLiveLinkLightBlueprintData


Facility structure to handle light data in blueprint

**属性**:

- `FLiveLinkLightFrameData FrameData [Dynamic data that can change every frame]`
- `FLiveLinkLightStaticData StaticData [Static data that should not change every frame]`


**方法**:

- `FLiveLinkLightBlueprintData& opAssign(FLiveLinkLightBlueprintData Other)`

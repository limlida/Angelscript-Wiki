### FRetargetSpeedPlantingSettings

- `FName SpeedCurveName [The name of the curve on the source animation that contains the speed of the end effector bone.]`
- `FName TargetChainName [The name of the target chain to plant the IK on.
NOTE: this chain must have an IK Goal assigned to it!]`

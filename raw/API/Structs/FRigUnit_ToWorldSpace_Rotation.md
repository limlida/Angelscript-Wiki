### FRigUnit_ToWorldSpace_Rotation


Converts a rotation from rig (global) space to world space

**属性**:

- `FQuat Value [The input rotation in global / rig space]`
- `FQuat World [The result rotation in world space]`


**方法**:

- `FRigUnit_ToWorldSpace_Rotation& opAssign(FRigUnit_ToWorldSpace_Rotation Other)`

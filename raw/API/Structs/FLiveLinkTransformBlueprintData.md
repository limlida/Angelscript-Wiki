### FLiveLinkTransformBlueprintData


Facility structure to handle transform data in blueprint

**属性**:

- `FLiveLinkTransformFrameData FrameData [Dynamic data that can change every frame]`
- `FLiveLinkTransformStaticData StaticData [Static data that should not change every frame]`


**方法**:

- `FLiveLinkTransformBlueprintData& opAssign(FLiveLinkTransformBlueprintData Other)`

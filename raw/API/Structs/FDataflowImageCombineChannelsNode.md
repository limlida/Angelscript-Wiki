### FDataflowImageCombineChannelsNode


Combine channels into a single RGBA image
Outputs are single channel images

**属性**:

- `FLinearColor OverrideColor []`
- `EDataflowImageResolution Resolution [resolution of the output image if the resolution option is set to user defined]`
- `EDataflowImageCombineResolutionOption ResolutionOptions [resolution option]`
- `bool bActive [Used to calculate ContainerElementOffsets]`
- `bool bOverrideColor []`


**方法**:

- `FDataflowImageCombineChannelsNode& opAssign(FDataflowImageCombineChannelsNode Other)`

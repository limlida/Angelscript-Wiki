### FRigVMFunction_VisualLogArrow


Logs arrow - recording for Visual Logs has to be enabled to record this data

**属性**:

- `float32 ArrowHeadSize [The size of the arrow head]`
- `FName Category [The category of the logged text]`
- `FRigVMDebugDrawSettings DebugDrawSettings`
- `FRigVMExecutePin ExecutePin [* This property is used to chain multiple mutable nodes together]`
- `FLinearColor ObjectColor [The color of the logged object]`
- `FVector SegmentEnd [The end of the arrow]`
- `FVector SegmentStart [The start of the arrow]`
- `FString Text [The text to log]`


**方法**:

- `FRigVMFunction_VisualLogArrow& opAssign(FRigVMFunction_VisualLogArrow Other)`

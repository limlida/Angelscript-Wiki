### FPhysicsControlNamedModifierParameters


Data that can be used to parameterize(modify / update) a modifier

**属性**:

- `FPhysicsControlModifierSparseData Data`
- `FName Name [The name of the modifier (or set of modifiers) to update]`


**方法**:

- `FPhysicsControlNamedModifierParameters& opAssign(FPhysicsControlNamedModifierParameters Other)`

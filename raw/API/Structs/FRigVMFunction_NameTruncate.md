### FRigVMFunction_NameTruncate


Returns the left or right most characters from the string chopping the given number of characters from the start or the end

**属性**:

- `FName Chopped [the part of the name that has been chopped off]`
- `int Count [Number of characters to remove from left or right]`
- `bool FromEnd [if set to true the characters will be removed from the end]`
- `FName Name`
- `FName Remainder [the part of the string without the chopped characters]`


**方法**:

- `FRigVMFunction_NameTruncate& opAssign(FRigVMFunction_NameTruncate Other)`

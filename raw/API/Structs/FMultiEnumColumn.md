### FMultiEnumColumn


A column which filters rows using an Enum variable, where rows pass if the enum is one of any of the checked values.

**属性**:

- `FChooserMultiEnumRowData DefaultRowValue [DefaultRowValue will be assigned to cells when new rows are created]`
- `FInstancedStruct InputValue [The Enum property this column will filter based on]`
- `bool bDisabled []`


**方法**:

- `FMultiEnumColumn& opAssign(FMultiEnumColumn Other)`

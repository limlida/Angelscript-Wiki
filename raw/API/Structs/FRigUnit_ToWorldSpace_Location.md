### FRigUnit_ToWorldSpace_Location


Converts a position / location from rig (global) space to world space

**属性**:

- `FVector Value [The input position / location in global / rig space]`
- `FVector World [The result position / location in world space]`


**方法**:

- `FRigUnit_ToWorldSpace_Location& opAssign(FRigUnit_ToWorldSpace_Location Other)`

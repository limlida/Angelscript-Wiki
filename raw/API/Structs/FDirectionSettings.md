### FDirectionSettings


Main direction settings configuration

**属性**:

- `EDirectionMode DirectionMode [Primary direction control mode]`
- `float32 DirectionOffsetAngle [Global direction offset angle]`
- `int DirectionSidesCount [Number of fixed direction sides]`
- `FDirectionMasterInputSettings InputSettings [Nested settings structures]`
- `FVisualSettings VisualSettings`


**方法**:

- `FDirectionSettings& opAssign(FDirectionSettings Other)`

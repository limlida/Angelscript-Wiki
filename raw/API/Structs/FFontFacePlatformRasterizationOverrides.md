### FFontFacePlatformRasterizationOverrides


Remapping of rasterization modes

**属性**:

- `EFontRasterizationMode MsdfOverride [Rasterization mode to be used instead of Sharp (Multi-Channel SDF)]`
- `EFontRasterizationMode SdfApproximationOverride [Rasterization mode to be used instead of Fast (Approximate SDF)]`
- `EFontRasterizationMode SdfOverride [Rasterization mode to be used instead of Smooth (Plain SDF)]`


**方法**:

- `FFontFacePlatformRasterizationOverrides& opAssign(FFontFacePlatformRasterizationOverrides Other)`

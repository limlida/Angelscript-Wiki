### FRigUnit_SetVectorAnimationChannel


Set Vector Channel is used to set a control's animation channel value

**属性**:

- `FName Channel [The name of the animation channel to retrieve the value for.]`
- `FName Control [The name of the Control to retrieve the value for.]`
- `FRigVMExecutePin ExecutePin`
- `FVector Value [The new value of the animation channel]`
- `bool bInitial [If set to true the initial value will be returned]`


**方法**:

- `FRigUnit_SetVectorAnimationChannel& opAssign(FRigUnit_SetVectorAnimationChannel Other)`

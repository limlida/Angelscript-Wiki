### FRigUnit_AddAvailableSpaces


Adds available spaces for a given control. This is used for the space switching widget.

**属性**:

- `FRigElementKey Control [* The available spaces to add for a given control]`
- `FRigVMExecutePin ExecutePin [* This property is used to chain multiple mutable units together]`
- `TArray<FRigElementKeyWithLabel> Spaces [* The spaces to add]`


**方法**:

- `FRigUnit_AddAvailableSpaces& opAssign(FRigUnit_AddAvailableSpaces Other)`

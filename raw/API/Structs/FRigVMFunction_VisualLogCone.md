### FRigVMFunction_VisualLogCone


Logs cone shape - recording for Visual Logs has to be enabled to record this data

**属性**:

- `float32 Angle [The angle of the cone]`
- `FName Category [The category of the logged text]`
- `FRigVMDebugDrawSettings DebugDrawSettings`
- `FVector Direction [The direction of the cone]`
- `FRigVMExecutePin ExecutePin [* This property is used to chain multiple mutable nodes together]`
- `float32 Length [The length of the cone]`
- `FLinearColor ObjectColor [The color of the logged object]`
- `FVector Origin [The origin of the cone]`
- `FString Text [The text to log]`
- `bool bWireframe [Whether to display as wireframe]`


**方法**:

- `FRigVMFunction_VisualLogCone& opAssign(FRigVMFunction_VisualLogCone Other)`

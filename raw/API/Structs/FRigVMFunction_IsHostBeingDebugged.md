### FRigVMFunction_IsHostBeingDebugged


Returns true if a graph is run with its asset editor open. This is editor only - in shipping it always returns false.

**属性**:

- `bool Result [True if the graph is currently open in the asset editor]`


**方法**:

- `FRigVMFunction_IsHostBeingDebugged& opAssign(FRigVMFunction_IsHostBeingDebugged Other)`

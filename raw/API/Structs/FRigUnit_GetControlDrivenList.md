### FRigUnit_GetControlDrivenList


GetControlDrivenList is used to retrieve the list of affected controls of an indirect control

**属性**:

- `FName Control [The name of the Control to get the list for]`
- `TArray<FRigElementKey> Driven [The list of affected controls]`


**方法**:

- `FRigUnit_GetControlDrivenList& opAssign(FRigUnit_GetControlDrivenList Other)`

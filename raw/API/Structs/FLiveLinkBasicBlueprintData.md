### FLiveLinkBasicBlueprintData


Facility structure to handle base data in blueprint

**属性**:

- `FLiveLinkBaseFrameData FrameData [Dynamic data that can change every frame]`
- `FLiveLinkBaseStaticData StaticData [Static data that should not change every frame]`


**方法**:

- `FLiveLinkBasicBlueprintData& opAssign(FLiveLinkBasicBlueprintData Other)`

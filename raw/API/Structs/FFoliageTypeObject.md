### FFoliageTypeObject


A wrapper struct used to allow the use of either FoliageType assets or FoliageType blueprint classes

**属性**:

- `UObject FoliageTypeObject [The foliage type that will be spawned by the procedural foliage simulation]`


**方法**:

- `FFoliageTypeObject& opAssign(FFoliageTypeObject Other)`

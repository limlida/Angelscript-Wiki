### FObjectColumn


A column which filters rows by an input Object to specified Objects for each row.

**属性**:

- `FChooserObjectRowData DefaultRowValue [DefaultRowValue will be assigned to cells when new rows are created]`
- `FInstancedStruct InputValue [The Object reference property this column will filter based on]`
- `bool bDisabled []`


**方法**:

- `FObjectColumn& opAssign(FObjectColumn Other)`

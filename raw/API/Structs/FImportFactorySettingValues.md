### FImportFactorySettingValues


Holds FProperty names and values to customize factory settings

**属性**:

- `FString SettingName [Name of the property to change.  Nested settings can be modified using "Outer.Property]`
- `FString Value [Value to import for the specified property.]`


**方法**:

- `FImportFactorySettingValues& opAssign(FImportFactorySettingValues Other)`

### FRigidBodyControlTarget


A single target for a control, which may be defined as an offset from the (implicit) animation target.

**属性**:

- `FRotator TargetOrientation [The target orientation of the child body, relative to the parent body]`
- `FVector TargetPosition [The target position of the child body, relative to the parent body]`


**方法**:

- `FRigidBodyControlTarget& opAssign(FRigidBodyControlTarget Other)`

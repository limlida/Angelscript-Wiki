### FSwarmDebugOptions


Debug options for Swarm

**属性**:

- `bool bDistributionEnabled [If true, Swarm will distribute jobs.
If false, only the local machine will execute the jobs.]`
- `bool bForceContentExport [If true, Swarm will force content to re-export rather than using the cached version.
If false, Swarm will attempt to use the cached version.]`


**方法**:

- `FSwarmDebugOptions& opAssign(FSwarmDebugOptions Other)`

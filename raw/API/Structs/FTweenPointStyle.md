### FTweenPointStyle


**属性**:

- `FVector2D HitTestSize [The size of the hit box against which hit tests are made for this point (so the user does not need to click them pixel perfectly).]`
- `FSlateBrush Hovered [Brush when a point on the bar is hovered]`
- `FSlateBrush Normal [Brush when a point on the bar is not hovered or pressed]`
- `FSlateBrush PassedPoint [Brush when the slider has passed a point]`
- `FSlateBrush Pressed [Brush when a point on the bar is pressed]`


**方法**:

- `FTweenPointStyle& opAssign(FTweenPointStyle Other)`

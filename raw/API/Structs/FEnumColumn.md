### FEnumColumn


A column that filters rows based on the value of an Enum, with Equal Not Equal, or Any as cell comparison options.

**属性**:

- `FChooserEnumRowData DefaultRowValue [DefaultRowValue will be assigned to cells when new rows are created]`
- `FInstancedStruct InputValue [The Enum property this column will filter based on]`
- `bool bDisabled []`


**方法**:

- `FEnumColumn& opAssign(FEnumColumn Other)`

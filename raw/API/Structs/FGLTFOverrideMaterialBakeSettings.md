### FGLTFOverrideMaterialBakeSettings


**属性**:

- `TextureFilter Filter [Overrides the default filtering mode used when sampling the baked out texture.]`
- `FGLTFMaterialBakeSize Size [Overrides default size of the baked out texture.]`
- `TextureAddress Tiling [Overrides the default addressing mode used when sampling the baked out texture.]`
- `bool bOverrideFilter [If enabled, default filtering mode will be overridden by the corresponding property.]`
- `bool bOverrideSize [If enabled, default size will be overridden by the corresponding property.]`
- `bool bOverrideTiling [If enabled, default addressing mode will be overridden by the corresponding property.]`


**方法**:

- `FGLTFOverrideMaterialBakeSettings& opAssign(FGLTFOverrideMaterialBakeSettings Other)`

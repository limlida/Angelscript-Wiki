### FDebugFloatHistory


Structure for recording float values and displaying them as an Histogram through DrawDebugFloatHistory.

**属性**:

- `int MaxSamples [Max Samples to record.]`
- `float32 MaxValue [Max value to record.]`
- `float32 MinValue [Min value to record.]`
- `bool bAutoAdjustMinMax [Auto adjust Min/Max as new values are recorded?]`


**方法**:

- `FDebugFloatHistory& opAssign(FDebugFloatHistory Other)`

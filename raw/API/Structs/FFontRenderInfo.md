### FFontRenderInfo


Information used in font rendering

**属性**:

- `FDepthFieldGlowInfo GlowInfo [Depth field glow parameters (only usable if font was imported with a depth field)]`
- `bool bClipText [Whether to clip text]`
- `bool bEnableShadow [Whether to turn on shadowing]`


**方法**:

- `FFontRenderInfo& opAssign(FFontRenderInfo Other)`

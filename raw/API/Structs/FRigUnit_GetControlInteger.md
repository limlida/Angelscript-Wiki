### FRigUnit_GetControlInteger


GetControlFloat is used to retrieve a single Integer from a hierarchy.

**属性**:

- `FName Control [The name of the Control to retrieve the Integer for.]`
- `int IntegerValue [The current value of the control.]`
- `int Maximum [The maximum value of the control.]`
- `int Minimum [The minimum value of the control.]`


**方法**:

- `FRigUnit_GetControlInteger& opAssign(FRigUnit_GetControlInteger Other)`

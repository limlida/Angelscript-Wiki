### FColorGradingSpinBoxStyle


Represents the appearance of a color grading spin box

**属性**:

- `FSlateBrush ActiveBorderBrush [Brush used to draw the border of the spinbox when it's in active use by the user]`
- `FSlateBrush BorderBrush [Brush used to draw the border of the spinbox]`
- `FSlateBrush HoveredBorderBrush [Brush used to draw the border of the spinbox when it's hovered over]`
- `FSlateBrush SelectorBrush [Brush used to draw the selector indicating the current value]`
- `float32 SelectorWidth [Width of the selector]`


**方法**:

- `FColorGradingSpinBoxStyle& opAssign(FColorGradingSpinBoxStyle Other)`

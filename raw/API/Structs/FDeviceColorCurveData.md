### FDeviceColorCurveData


Data required for setting the Input Device Color

**属性**:

- `UCurveLinearColor DeviceColorCurve [The color the device light should be]`
- `bool bEnable [True if the light should be enabled at all]`
- `bool bResetAfterCompletion [If true, the light color will be reset to "off" after the curve values are finished evaluating.]`


**方法**:

- `FDeviceColorCurveData& opAssign(FDeviceColorCurveData Other)`

### FOptimusDeformerInstanceComponentBinding


Defines a binding between a component provider in the graph and an actor component in the component hierarchy on
the actor whose deformable component we're bound to.

**属性**:

- `FName ComponentName [Component name to bind. This should be sanitized before storage.]`


**方法**:

- `FOptimusDeformerInstanceComponentBinding& opAssign(FOptimusDeformerInstanceComponentBinding Other)`

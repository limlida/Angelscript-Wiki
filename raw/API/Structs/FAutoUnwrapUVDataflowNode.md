### FAutoUnwrapUVDataflowNode


Auto unwrap UVs for a specific UV channel

**属性**:

- `int GutterSize [Approximate space to leave between UV islands, measured in texels for 512x512 texture]`
- `FLinearColor OverrideColor []`
- `int UVChannel [UV channel to unwrap into ( 0 by default )]`
- `bool bActive [Used to calculate ContainerElementOffsets]`
- `bool bOverrideColor []`


**方法**:

- `FAutoUnwrapUVDataflowNode& opAssign(FAutoUnwrapUVDataflowNode Other)`

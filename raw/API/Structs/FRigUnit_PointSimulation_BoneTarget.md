### FRigUnit_PointSimulation_BoneTarget


**属性**:

- `FName Bone [The name of the bone to map]`
- `int PrimaryAimPoint [The index of the point to use for aiming the primary axis.
Use -1 to indicate that you don't want to aim the bone.]`
- `int SecondaryAimPoint [The index of the point to use for aiming the secondary axis.
Use -1 to indicate that you don't want to aim the bone.]`
- `int TranslationPoint [The index of the point to use for translation]`


**方法**:

- `FRigUnit_PointSimulation_BoneTarget& opAssign(FRigUnit_PointSimulation_BoneTarget Other)`

### FClothConstraintSetupNv


Container for a constraint setup, these can be horizontal, vertical, shear and bend.

**属性**:

- `float32 CompressionLimit [The hard limit on how far this constraint can compress]`
- `float32 Stiffness [How stiff this constraint is, this affects how closely it will follow the desired position]`
- `float32 StiffnessMultiplier [A multiplier affecting the above value]`
- `float32 StretchLimit [The hard limit on how far this constraint can stretch]`


**方法**:

- `FClothConstraintSetupNv& opAssign(FClothConstraintSetupNv Other)`

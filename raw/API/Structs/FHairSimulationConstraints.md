### FHairSimulationConstraints


**属性**:

- `float32 BendDamping [Damping for the bend constraint between 0 and 1]`
- `float32 BendStiffness [Stiffness for the bend constraint in GPa]`
- `float32 CollisionRadius [Radius that will be used for the collision detection against the physics asset]`
- `float32 KineticFriction [Kinetic friction used for collision against the physics asset]`
- `float32 StaticFriction [Static friction used for collision against the physics asset]`
- `float32 StrandsViscosity [Viscosity parameter between 0 and 1 that will be used for self collision]`
- `float32 StretchDamping [Damping for the stretch constraint between 0 and 1]`
- `float32 StretchStiffness [Stiffness for the stretch constraint in GPa]`


**方法**:

- `FHairSimulationConstraints& opAssign(FHairSimulationConstraints Other)`

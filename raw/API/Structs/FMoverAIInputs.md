### FMoverAIInputs


Data block containing ROV Velocity that is networked from server to clients.
Also supports rewind/resimulation of the data.

**属性**:

- `FVector RVOVelocityDelta [ROV Velocity calculated on the Server]`


**方法**:

- `FMoverAIInputs& opAssign(FMoverAIInputs Other)`

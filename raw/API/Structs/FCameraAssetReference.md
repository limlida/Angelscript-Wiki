### FCameraAssetReference


A structure holding a reference to a camera asset, along with the interface parameter
override values for any of its camera rigs.

**属性**:

- `UCameraAsset CameraAsset [The referenced camera asset.]`
- `FInstancedOverridablePropertyBag Parameters [The camera asset's parameters.]`


**方法**:

- `FCameraAssetReference& opAssign(FCameraAssetReference Other)`

### FRigUnit_SequenceExecution


Allows for a single execution pulse to trigger a series of events in order.

**属性**:

- `FControlRigExecuteContext A [The execution result A]`
- `FControlRigExecuteContext B [The execution result B]`
- `FControlRigExecuteContext C [The execution result C]`
- `FControlRigExecuteContext D [The execution result D]`
- `FControlRigExecuteContext ExecuteContext [The execution input]`


**方法**:

- `FRigUnit_SequenceExecution& opAssign(FRigUnit_SequenceExecution Other)`

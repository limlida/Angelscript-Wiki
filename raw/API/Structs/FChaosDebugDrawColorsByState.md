### FChaosDebugDrawColorsByState


Structure holding the settings using to debug draw Particles shape based on their state on the Chaos Visual Debugger

**属性**:

- `FColor DynamicColor [Color used for dynamic particles]`
- `FColor KinematicColor [Color used for kinematic particles]`
- `FColor SleepingColor [Color used for sleeping particles]`
- `FColor StaticColor [Color used for static particles]`


**方法**:

- `FChaosDebugDrawColorsByState& opAssign(FChaosDebugDrawColorsByState Other)`

### FIKRigBodyMoverSettings


**属性**:

- `float32 PositionAlpha [Blend the translational effect of this solver on/off. Range is 0-1. Default is 1.0.]`
- `float32 PositionNegativeX [Multiply the NEGATIVE X translation. Range is 0-1. Default is 1.0.]`
- `float32 PositionNegativeY [Multiply the NEGATIVE Y translation. Range is 0-1. Default is 1.0.]`
- `float32 PositionNegativeZ [/ Multiply the NEGATIVE Z translation. Range is 0-1. Default is 1.0.]`
- `float32 PositionPositiveX [Multiply the POSITIVE X translation. Range is 0-1. Default is 1.0.]`
- `float32 PositionPositiveY [Multiply the POSITIVE Y translation. Range is 0-1. Default is 1.0.]`
- `float32 PositionPositiveZ [Multiply the POSITIVE Z translation. Range is 0-1. Default is 1.0.]`
- `float32 RotateXAlpha [Blend the X-axis rotational effect on/off. Range is 0-1. Default is 1.0.]`
- `float32 RotateYAlpha [Blend the Y-axis rotational effect on/off. Range is 0-1. Default is 1.0.]`
- `float32 RotateZAlpha [Blend the Z-axis rotational effect on/off. Range is 0-1. Default is 1.0.]`
- `float32 RotationAlpha [Blend the total rotational effect on/off. Range is 0-1. Default is 1.0.]`
- `FName StartBone [The target bone to move with the effectors.]`


**方法**:

- `FIKRigBodyMoverSettings& opAssign(FIKRigBodyMoverSettings Other)`

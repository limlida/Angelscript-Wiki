### FNarrativeItemStat


A stat that can be shown in the item preview window

**属性**:

- `FText StatDisplayName [The stats display name.]`
- `FText StatTooltip [Tooltip the stat will show when hovered]`
- `FString StringVariable [The backing string variable for the stat - implemented using GetStringVariable function that can be overriden in Blueprints!]`


**方法**:

- `FNarrativeItemStat& opAssign(FNarrativeItemStat Other)`

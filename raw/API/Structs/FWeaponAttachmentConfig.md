### FWeaponAttachmentConfig


Usually one of these would be defined per slot the weapon is in. HipL/HipR attach in different places etc.

**属性**:

- `FTransform Offset [Any relative offset we want to apply]`
- `FName SocketName [The bone or socket we want the attachment to equip to]`


**方法**:

- `FWeaponAttachmentConfig& opAssign(FWeaponAttachmentConfig Other)`

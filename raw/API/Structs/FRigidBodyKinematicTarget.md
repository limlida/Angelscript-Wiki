### FRigidBodyKinematicTarget


A single kinematic target, which will be applied in the KinematicTargetSpace set in the relevant body modifier.

**属性**:

- `FRotator TargetOrientation [The target orientation of the body]`
- `FVector TargetPosition [The target position of the body]`


**方法**:

- `FRigidBodyKinematicTarget& opAssign(FRigidBodyKinematicTarget Other)`

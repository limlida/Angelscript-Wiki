### FRigUnit_GetItemModuleName


Returns the namespace of a given item. This may be an empty string if the item doesn't have a namespace.

**属性**:

- `bool IsPartOfModule [* True if the item is part of a module]`
- `FRigElementKey Item [* The key of the item to return the module for]`
- `FString Module [* The name of the module of the given item]`


**方法**:

- `FRigUnit_GetItemModuleName& opAssign(FRigUnit_GetItemModuleName Other)`

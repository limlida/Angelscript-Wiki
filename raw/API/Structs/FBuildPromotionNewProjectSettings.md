### FBuildPromotionNewProjectSettings


Holds settings for the new project stage of the build promotion test

**属性**:

- `FDirectoryPath NewProjectFolderOverride [The path for the new project]`
- `FString NewProjectNameOverride [The name of the project *]`


**方法**:

- `FBuildPromotionNewProjectSettings& opAssign(FBuildPromotionNewProjectSettings Other)`

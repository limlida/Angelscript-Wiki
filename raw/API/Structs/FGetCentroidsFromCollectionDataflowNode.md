### FGetCentroidsFromCollectionDataflowNode


Gets centroids of pieces from a Collection

**属性**:

- `FLinearColor Color [Point color]`
- `FLinearColor OverrideColor []`
- `FRuntimeFloatCurve PointSize [Float curve to control point size by level]`
- `float32 Size [Point size]`
- `bool bActive [Used to calculate ContainerElementOffsets]`
- `bool bColorByLevel [Control point color by level or setting]`
- `bool bOverrideColor []`
- `bool bSizeByLevel [Control the point size by level or setting]`


**方法**:

- `FGetCentroidsFromCollectionDataflowNode& opAssign(FGetCentroidsFromCollectionDataflowNode Other)`

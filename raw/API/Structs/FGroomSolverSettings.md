### FGroomSolverSettings


Solver settings that will be used in dataflow/deformergraph

**属性**:

- `float32 MaxLODDistance [Maximum LOD distance (if distance between the component and the views is higher that this threshold, no simulation)]`
- `float32 MaxPointsCount [Maximum number of points the solver is going to simulate]`
- `float32 MinLODDistance [Minimum LOD distance (if distance between the component and the views is lower that this threshold, no simulation)]`


**方法**:

- `FGroomSolverSettings& opAssign(FGroomSolverSettings Other)`

### FWriteStringToFile


Write a string to a file

**属性**:

- `FString FileContents [Contents of the file to write]`
- `FString FilePath [Where file should be written on disk]`
- `FLinearColor OverrideColor []`
- `bool bActive [Used to calculate ContainerElementOffsets]`
- `bool bOverrideColor []`


**方法**:

- `FWriteStringToFile& opAssign(FWriteStringToFile Other)`

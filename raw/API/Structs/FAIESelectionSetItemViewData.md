### FAIESelectionSetItemViewData


View Data for the set, contains location and color in the UI

**属性**:

- `FLinearColor Color [The Color of the Set]`
- `int Row [The logical row the set should be in, basically it's order. This is an internal order for just one set]`


**方法**:

- `FAIESelectionSetItemViewData& opAssign(FAIESelectionSetItemViewData Other)`

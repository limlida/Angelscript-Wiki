### FBoxSphereBounds3d


A bounding box and bounding sphere with the same origin.
@note The full C++ class is located here : Engine\Source\Runtime\Core\Public\Math\BoxSphereBounds.h

**属性**:

- `FVector3d BoxExtent [Holds the extent of the bounding box, which is half the size of the box in 3D space]`
- `FVector3d Origin [Holds the origin of the bounding box and sphere.]`
- `float SphereRadius [Holds the radius of the bounding sphere.]`

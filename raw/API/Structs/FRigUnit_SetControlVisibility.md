### FRigUnit_SetControlVisibility


SetControlVisibility is used to change the visibility on a control at runtime

**属性**:

- `FRigVMExecutePin ExecutePin [* This property is used to chain multiple mutable units together]`
- `FRigElementKey Item [The name of the Control to set the visibility for.]`
- `FString Pattern [If the ControlName is set to None this can be used to look for a series of Controls]`
- `bool bVisible [The visibility to set for the control]`


**方法**:

- `FRigUnit_SetControlVisibility& opAssign(FRigUnit_SetControlVisibility Other)`

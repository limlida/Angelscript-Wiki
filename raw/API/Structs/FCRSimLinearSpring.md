### FCRSimLinearSpring


**属性**:

- `float32 Coefficient [The power of this spring]`
- `float32 Equilibrium [The rest length of this spring.
A value of lower than zero indicates that the equilibrium
should be based on the current distance of the two subjects.]`
- `int SubjectA [The first point affected by this spring]`
- `int SubjectB [The second point affected by this spring]`


**方法**:

- `FCRSimLinearSpring& opAssign(FCRSimLinearSpring Other)`

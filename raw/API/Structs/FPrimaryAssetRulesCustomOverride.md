### FPrimaryAssetRulesCustomOverride


Apply primary asset rules to groups of primary assets, using type + filter directory or string

**属性**:

- `FDirectoryPath FilterDirectory [Will only apply to files in this directory]`
- `FString FilterString [Game-specific string defining which assets to apply this to]`
- `FPrimaryAssetType PrimaryAssetType [Which type to apply rules for]`
- `FPrimaryAssetRules Rules [What to overrides the rules with]`


**方法**:

- `FPrimaryAssetRulesCustomOverride& opAssign(FPrimaryAssetRulesCustomOverride Other)`

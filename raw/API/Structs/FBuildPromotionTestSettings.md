### FBuildPromotionTestSettings


Holds settings for the editor build promotion test

**属性**:

- `FFilePath DefaultStaticMeshAsset [Default static mesh asset to apply materials to *]`
- `FBuildPromotionImportWorkflowSettings ImportWorkflow [Import workflow settings *]`
- `FBuildPromotionNewProjectSettings NewProjectSettings [New project settings *]`
- `FBuildPromotionOpenAssetSettings OpenAssets [Open assets settings *]`
- `FFilePath SourceControlMaterial [Material to modify for the content browser step *]`


**方法**:

- `FBuildPromotionTestSettings& opAssign(FBuildPromotionTestSettings Other)`

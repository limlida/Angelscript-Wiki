### FBlueprintEditorPromotionSettings


Holds settings for the blueprint editor build promotion tests

**属性**:

- `FFilePath DefaultParticleAsset [Default particle asset to use for tests]`
- `FFilePath FirstMeshPath [The starting mesh for the blueprint *]`
- `FFilePath SecondMeshPath [The mesh to set on the blueprint after the delay *]`


**方法**:

- `FBlueprintEditorPromotionSettings& opAssign(FBlueprintEditorPromotionSettings Other)`

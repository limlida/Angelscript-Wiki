### FAudioInputDeviceChannelProperty


This class is used by Microphone sources to track the currently selected channel. It aslo
contains a local cache of the max channel count for the currently selected audio device.

**属性**:

- `int AudioInputDeviceChannel [The currently selected channel from the details panel for this source]`


**方法**:

- `FAudioInputDeviceChannelProperty& opAssign(FAudioInputDeviceChannelProperty Other)`

### FDangerDetectionSharedFragment


**属性**:

- `float32 MinSpeed [The min speed to consider a dangerous entity.]`
- `float32 ObstacleQueryExtent [The range we check for incoming dangerous obstacles]`
- `float32 ObstacleRadiusMultiplier [The radius multiplier for incoming dangerous obstacles. This gives peds more room to react to incoming dangerous obstacles]`
- `float32 TimeToCollisionEvent [Time in seconds to trigger IncomingDangerousEntity() on ped NPCs when there is an incoming dangerous vehicle]`


**方法**:

- `FDangerDetectionSharedFragment& opAssign(FDangerDetectionSharedFragment Other)`

### FAssignMaterialInterfaceToCollectionDataflowNode


Assign material to a set of face in a geometry collection

**属性**:

- `UMaterialInterface Material [Material to assign to the selection]`
- `FLinearColor OverrideColor []`
- `bool bActive [Used to calculate ContainerElementOffsets]`
- `bool bMergeDuplicateMaterials [If true, detect duplicate in the material array and only add the material in the array if it does not yet exists]`
- `bool bOverrideColor []`


**方法**:

- `FAssignMaterialInterfaceToCollectionDataflowNode& opAssign(FAssignMaterialInterfaceToCollectionDataflowNode Other)`

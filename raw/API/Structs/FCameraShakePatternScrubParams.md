### FCameraShakePatternScrubParams


Parameters for scrubbing a camera shake.

**属性**:

- `float32 AbsoluteTime [The time to scrub to]`
- `float32 DynamicScale [The dynamic scale being passed down from the camera manger for the next update]`
- `FMinimalViewInfo POV [The current view that this camera shake should modify]`
- `float32 ShakeScale [The base scale for this shake]`


**方法**:

- `FCameraShakePatternScrubParams& opAssign(FCameraShakePatternScrubParams Other)`

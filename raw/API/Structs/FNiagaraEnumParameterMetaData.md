### FNiagaraEnumParameterMetaData


Defines override data for enum parameters displayed in the UI.

**属性**:

- `FLinearColor ColorOverride []`
- `UTexture2D IconOverride [If specified, this icon will be used for the given enum entry. If OverrideName isn't empty, the icon takes priority.]`
- `FName OverrideName [If specified, this name will be used for the given enum entry. Useful for shortening names.]`
- `bool bUseColorOverride []`


**方法**:

- `FNiagaraEnumParameterMetaData& opAssign(FNiagaraEnumParameterMetaData Other)`

### FAlphaBlend


Alpha Blend class that supports different blend options as well as custom curves

**属性**:

- `EAlphaBlendOption BlendOption [Type of blending used (Linear, Cubic, etc.)]`
- `float32 BlendTime [Blend Time]`
- `UCurveFloat CustomCurve [If you're using Custom BlendOption, you can specify curve]`


**方法**:

- `FAlphaBlend& opAssign(FAlphaBlend Other)`

### FWidgetSnapshotRequest


Implements a message that is used to request a widget snapshot from a remote target.

**属性**:

- `FGuid SnapshotRequestId [The request ID of this snapshot (used to identify the correct response from the target)]`
- `FGuid TargetInstanceId [The instance ID of the remote target we want to receive a snapshot from]`


**方法**:

- `FWidgetSnapshotRequest& opAssign(FWidgetSnapshotRequest Other)`

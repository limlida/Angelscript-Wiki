## 概述
本文档是 UnrealEngine-Angelscript 官方文档的主页，介绍了该项目的背景、特性和开发历程。

## 核心要点

### 项目简介
- **开发者**：Hazelight 工作室（《双人成行》、《分裂幻想》的开发商）
- **用途**：将 Angelscript 脚本语言集成到 Unreal Engine 中
- **成熟度**：自 2018 年起在生产环境使用，30+ 开发者日常使用
- **代码规模**：《分裂幻想》使用超过 1,700,000 行 Angelscript 代码，分布在 16,000+ 脚本文件中

### 核心优势
1. **快速迭代**：脚本热重载，无需重启编辑器
2. **改进协作**：程序员和设计师在同一系统中工作
3. **高性能**：接近 C++ 性能，可转译为 C++ 用于发布版本

### 主要特性
- **简化语法**：类似 C++ 但更简洁，避免常见陷阱
- **完整编辑器支持**：VS Code 扩展支持自动补全、错误诊断、重命名、引用查找
- **与 Blueprint 无缝集成**：可重写 Blueprint 事件，可作为 Blueprint 基类
- **调试支持**：通过 VS Code 调试脚本

### 平台支持
- 正式支持：Windows、PS5、Xbox Series X|S
- 其他平台：社区已成功构建，但需自行验证

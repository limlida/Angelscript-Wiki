## 概述
本文档介绍了如何安装和配置 UnrealEngine-Angelscript。

## 核心要点

### 安装步骤
1. **获取 Unreal Engine 源码访问权限**
   - 需要 Epic Games 提供的源代码访问权限
   - 访问 https://www.unrealengine.com/en-US/ue-on-github

2. **获取 UnrealEngine-Angelscript**
   - 从 GitHub Releases 下载预编译编辑器
   - 或克隆 GitHub 仓库自行编译
   - **重要**：如需使用额外插件，必须自行编译源码

3. **安装 VS Code 扩展**
   - 从 https://code.visualstudio.com/ 安装 VS Code
   - 从 Marketplace 安装 "Unreal Angelscript" 扩展

## 注意事项
- 该修改过的引擎与普通 Unreal 二进制插件不兼容
- 如需使用插件，必须从源码重新编译

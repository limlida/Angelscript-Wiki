## 概述
Angelscript 的类型系统与 C++ 有重要区别，主要是去掉了指针，简化了引用管理。

## 核心区别

### 指针 vs 引用

#### C++ 方式
```cpp
AActor* ActorPtr = GetActor();
ActorPtr->SetActorLocation(Location);
```

#### Angelscript 方式
```angelscript
AActor Actor = GetActor();
Actor.SetActorLocation(Location);  // 使用 . 而不是 ->
```

## 对象引用

### UObject 都是引用
```angelscript
// 所有 UObject 类型都是引用
AActor Actor = GetActor();
USceneComponent Comp = GetRootComponent();
UStaticMeshComponent Mesh = UStaticMeshComponent::Get(Actor);

// 使用 . 访问成员
Comp.SetWorldLocation(FVector::ZeroVector);
```

### 空值检查
```angelscript
AActor Target = GetTarget();

// 使用 nullptr 检查
if (Target != nullptr)
{
    Target.DoSomething();
}

// 或使用 IsValid（推荐）
if (IsValid(Target))
{
    Target.DoSomething();
}
```

## 结构体引用

### 值类型 vs 引用类型
- **UObject 及其子类**：引用类型
- **结构体（FVector、FRotator 等）**：值类型

### 结构体引用传递
```angelscript
// 使用 & 传递引用，允许修改
void ModifyVector(FVector& Vector)
{
    Vector.X += 10.0;
}

// 调用
FVector MyVec = FVector(1, 2, 3);
ModifyVector(MyVec);  // MyVec 现在被修改了
```

### Mixin 结构体
```angelscript
// 为结构体添加 Mixin 方法
mixin void SetToZero(FVector& Vector)
{
    Vector = FVector::ZeroVector;
}

// 使用
FVector MyVec = FVector(1, 2, 3);
MyVec.SetToZero();  // 修改了 MyVec
```

## 垃圾回收

### 自动 GC 管理
```angelscript
class AMyActor : AActor
{
    // 不需要 UPROPERTY() 来防止 GC
    // 所有对象引用自动被 GC 管理
    AActor CachedTarget;
    
    UFUNCTION()
    void CacheTarget(AActor Target)
    {
        CachedTarget = Target;  // 安全，自动 GC
    }
}
```

### 优势
- 无需手动管理引用
- 无需担心悬空指针
- 自动检测生命周期

## 浮点数类型

### 默认是 64-bit
```angelscript
// Angelscript 中 float 默认是 64-bit double
float Value = 1.0;  // 实际上是 double
```

### 显式类型
```angelscript
float32 SinglePrecision = 1.0f;  // 32-bit float
float64 DoublePrecision = 1.0;   // 64-bit double（与 float 相同）
```

## 其他类型简化

### 类型推断 (auto)
```angelscript
// 使用 auto 进行类型推断
auto Player = Cast<APlayerCharacter>(GetOwner());
auto Location = GetActorLocation();
auto Tags = GetGameplayTags();
```

### 容器简化
```angelscript
// TArray 简化为数组语法
TArray<int32> Numbers;
Numbers.Add(1);
Numbers.Add(2);

// TMap 简化为字典语法
TMap<FName, float> Stats;
Stats.Add(n"Health", 100.0);
Stats.Add(n"Mana", 50.0);
```

### 范围 for 循环
```angelscript
// 遍历数组
for (int32 Number : Numbers)
{
    Print(f"Number: {Number}");
}

// 遍历 Map
for (auto Kvp : Stats)
{
    Print(f"{Kvp.GetKey()}: {Kvp.GetValue()}");
}
```

### 类型转换
```angelscript
// C++ 风格的 Cast
APlayerCharacter Player = Cast<APlayerCharacter>(Actor);

if (Player != nullptr)
{
    Player.DoPlayerThing();
}

// 使用 as 关键字（如果支持）
// APlayerCharacter Player = Actor as APlayerCharacter;
```

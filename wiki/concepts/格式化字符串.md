## 概述
Angelscript 支持 Python 风格的格式化字符串（f-strings），允许直接在字符串中嵌入表达式。

## 基本用法

### 简单插值
```angelscript
// 传统方式
FString Message = "Player: " + PlayerName + ", Score: " + FString::FromInt(Score);

// f-string 方式
FString Message = f"Player: {PlayerName}, Score: {Score}";
Print(Message);
```

### 在 Print 中使用
```angelscript
// 直接在 Print 中使用
Print(f"Called from actor {GetName()} at location {ActorLocation}");
```

## 高级用法

### 表达式
```angelscript
// 支持数学表达式
Print(f"Health: {Health} / {MaxHealth} ({Health / MaxHealth * 100}%)");

// 支持函数调用
Print(f"Distance: {GetDistanceTo(Target)} meters");
```

### 格式说明符

#### 数字格式化
```angelscript
// 小数精度
Print(f"Three Decimals: {ActorLocation.Z :.3}");           // 123.456

// 前导零填充
Print(f"Extended to 10 digits: {400 :010d}");              // 0000000400

// 十六进制
Print(f"Hexadecimal: {20 :#x}");                           // 0x14

// 二进制
Print(f"Binary: {1574 :b}");                               // 11000100110

// 科学计数法
Print(f"Scientific: {12345.6789 :.2e}");                   // 1.23e+04
```

#### 显示表达式名
```angelscript
// 使用 = 显示表达式和值
Print(f"{DeltaSeconds = :.2f}");  // 输出: DeltaSeconds = 0.02
```

### 对齐
```angelscript
Print(f"{Text :>40}");           // 右对齐到 40 字符
Print(f"{Text :_<40}");          // 左对齐，用下划线填充
```

### 枚举格式化
```angelscript
// 默认显示完整信息
Print(f"{ESlateVisibility::Collapsed}");  
// 输出: ESlateVisibility::Collapsed (1)

// 使用 :n 只显示名称
Print(f"{ESlateVisibility::Collapsed :n}");
// 输出: Collapsed
```

### 多行字符串
```angelscript
FString MultiLine = f""
    Player: {PlayerName}
    Health: {CurrentHealth}/{MaxHealth}
    Position: {GetActorLocation()}
"";
```
